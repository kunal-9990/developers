Release history - Analytics

The following describes the features and improvements made in previous releases of Analytics (select Help | About to find Cloud's version number).

What's new - Analytics 26.0

The following describes the features and improvements made for Analytics in CaseWare Cloud 26.0 (select Help | About to find Cloud's version number).

QuickBooks Online import

Files can now be imported directly from QuickBooks Online. When this feature is selected, Analytics prompts the user to log into their QuickBooks Online account and select a file to be imported.

Desktop imports

Transaction details can now be imported from most major desktop accounting platforms to Analytics via the Desktop Utility Import Tool. For more information, see Formatting client data from desktop software for importing into CaseWare Cloud.

New test: Transaction summary by account number

The Analytics team has added a new test for Transformer tool datafiles, Transaction summary by account number. This test can be used to view transaction details for a date range beyond the current fiscal year. This test can be accessed from the Account Analysis section.

General improvements

Several improvements have been made for the latest release.

  • Test parameters now display as expected on test results for the SE Integration test.

  • The Transactions by Transaction Date profile chart is now a bubble chart. This increases clarity in data sets that span multiple years or encompass many transactions.

  • Fixed an issue where account or customer ranges wouldn’t display properly in Parameters on first attempt.

  • Fixed display issues in saved test results for Transformer files with multiple journal sources in the same category.

What's new - Analytics 25.0

The following describes the features and improvements for CaseWareCloud Analytics 25.0.

Upload speeds

The upload speed for .csv files has been greatly increased. A 1GB file upload would previously take 5-6 hours, and now takes around 6 minutes.

The Profile page

Users can now drill down on overview data for Customers, Suppliers, Fixed Assets and Inventory master files.

Interface improvements

  • The current entity and engagement name now both display next to the Cloud menu icon.

  • We’ve updated the File Upload dialog.

  • We’ve also redesigned the first two profile graphs for increased clarity.

General improvements

  • Fixed an issue where .csv data files with columns set to Ignore on import produced errors when running some tests.

  • Fixed an issue where the first record was not displaying when running the Monetary Unit Sampling test on Accounts Receivable data files.

What's new - Analytics 24.0

The following describes the features and improvements for CaseWareCloud Analytics 24.0.

Import from Xero Online Accounting

We've added support for Xero Online Accounting, enabling you to import client data from Xero directly to your Analytics engagement without having to export it as a .csv file first. Select Upload New File | Import File From Xero and log into your Xero account to select the file you want to import.

For more on importing from Xero, please see Import a client's financial data from third-party software for analysis.

Profile page improvements

You can now drill down to view detailed transactions from Profile charts by selecting any individual data point.

The way Profile charts are displayed for journal and ledger files has been improved. Charts for transaction values and transaction counts now display values horizontally and side-by-side for easy comparison, and outliers in the data should now be easier to identify.

Additionally, the Chart icons have been moved for greater accessibility.

Analytics engagement tracking

In order to better track your workflow, you can now assign Analytics engagements to staff members. You can assign an engagement when creating a file or when editing it, and you can also filter the Analytics page to see which engagements are assigned to your staff.

Additionally, you can now enter the following information when creating a new Analytics engagement:

  • Start Date

  • End Date

  • Deliverable Due

  • Budget Hours

  • Budget Amount

This will enable you to measure the progress of Analytics engagements on the Engagements page.

Test enhancements

When using the Entries with specific comments test with CSV files, you can now search for keywords in multiple columns. Additionally, the Detailed account analysis test now supports CSV files. To learn more, see Run tests on your client data.

General improvements

  • When a document contains empty columns, those columns are now hidden in test results.

  • Analytics will now prompt you to upgrade when running tests on older or unsupported Transformer files.

  • Fixed an issue where running tests on incompatible files could produce unexpected results. If a test is incompatible with a Transformer file, you can no longer run it.
