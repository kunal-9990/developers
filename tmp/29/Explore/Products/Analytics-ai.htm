CaseWare Analytics.AI

This feature is only available with CaseWare Analytics.AI, part of the CaseWare Cloud suite.

Learn more about Cloud.

CaseWare Analytics.AI is a risk-based transaction analysis tool that auditors can use to scan large transaction sets and find exceptions that might warrant further investigation.

Analytics.AI complements CaseWare IDEA by giving a high level summary to users. It reduces the time spent in transaction analysis by automatically performing multiple tests on the entire transaction set.

For more information, see What's new - Analytics.AI beta release.
