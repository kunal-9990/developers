Add a firm logo and letterhead

You can add your firmâ€™s logo and letterhead in the firm template. Once you add the logo, it automatically replaces the logo placeholders in the deliverables. Once you add the letterhead, it automatically displays in the PDF copy (print copy) of the deliverables.

Note: To access the firm template, you need to have the Settings Admin role. For more information, see Built-in roles and Assign security roles.

To add a logo and letterhead in the firm template:

  1. Go to the Cloud menu (), select Settings () | <your product name> | Letterhead/Logo.

  2. At Letterhead, select the file upload box to browse and add your firm logo.

    3. You can also drag and drop the file.

  3. At Logo, select the file upload box to browse and add your firm letterhead.

    4. You can also drag and drop the file.

You can select REMOVE to remove the added logo or letterhead and upload another one.
