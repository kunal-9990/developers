Set up the notes library

The Notes Library provides a pool of notes which allows users to easily find and add notes in the financial statements document. Notes are automatically organized in the library by Categories (note and note headings).

The library also features a search field, where users can enter a keyword to filter notes.

To set up the Notes Library:

  1. Add notes

  2. Add associations to notes (optional)

  3. Exclude notes

Add notes

Note headings help you create a hierarchical structure for the financial statements notes by creating a multilevel list. You can use note headings to categorize the financial statements notes.

Note: You can only define a three-level list (a sample note structure is available here).

To add a note:

  1. Select the NOTES TO THE FINANCIAL STATEMENTS area at the Document Map.

  2. Select Add () | Create Note to add a note.

    3. A new note area is also added in the document with a default title Note title. You can select the default title and change it.

To add a note heading:

  1. Select the NOTES TO THE FINANCIAL STATEMENTS at the Document Map.

  2. Select the note that you want to add a heading for, then select Add ().

  3. Select the available heading under Heading.

    4. A sub-level note is added in the document with a default title Note title. You can select the default title and change it.

You can add content to a note. Select Add (), then choose one of the following content types:

  • Text Section (see Add text)

  • Dynamic Table (see Set up dynamic tables)

  • Page Break (see Add a page break)

  • Statement of Cash Flows (see Set up the statement of Cash Flows)

    Note: To have this option available in the list of content types, you need to set up the Statement of Cash Flows first.

Add associations to notes (optional)

An association refers to the trial balance groups that a specific note is related to. Use associations to further categorize notes by associating them with specific trial balance groups.

To associate a note to a trial balance group:

  1. Select the note that you want to associate with a trial balance group.

  2. Select More actions () | Associations.

  3. Select Add Financial Groups () to populate the trial balance data.

  4. Choose a trial balance group to associate the note with.

  5. Select Save.

Exclude notes

When a note is excluded from the document, itâ€™s stored in the Notes Library. The Notes Library is only available if there are excluded notes in the document. To construct the Notes Library, you need to exclude existing notes.

To have a note automatically excluded from the document, set up visibility conditions for each note in the financial statements document. Once the conditions are met, the note is automatically excluded from the document and stored in the Notes Library.

To manually exclude a note:

  1. Go to the note that you want to exclude.

  2. Select More actions () | Exclude.

    3. The note is excluded from the document and stored in the Notes Library.
