Define groupings

Groupings provide context to the accounts that users import into the product. You need to define groupings to set a structure for trial balance accounts.

The balances in the financial statements document are organized and displayed based on the groupings you defined. If your users can import a trial balance into the product then you must define a grouping structure.

The following grouping structures are available:

  • Financial grouping - Resembles the layout of the Balance Sheet and Income statement. It supports:

    • generating the financial statements and notes.

    • performing Risk Assessment in the RMM report.

    • providing the appropriate level of detail for analysis graphs, materiality benchmarks, visibility conditions and leadsheets.

  • Tax grouping - Resembles the tax codes applicable for your market. Automapping between the Financial grouping and corresponding tax code is available.

You can manually add groupings or you can import them from an Excel file or a CSV file, or from Working Papers.
