Add visibility logic in a letter or a memo

Visibility settings provide built-in automation so that only the relevant items are available in your document. This is achieved by setting up visibility conditions for various items.

To learn more, see What are visibility settings?

Add visibility logic for the document

You can add visibility logic for the entire document.

To set visibility for the document:

  1. Select the document name.

    2. The Document Properties dialog displays.

  2. Select Edit Visibility to display the Visibility Settings dialog.

  3. Select Hide or Show from the blue drop-down list to specify the visibility action.

  4. Select all or any to determine whether all conditions must be met or one condition must be met for the visibility action (from Step 3) to apply.

  5. Select Add Condition.

  6. Choose a condition type from the list, then specify the condition parameters.

    7. See Visibility Conditions for a detailed description of the available condition types and parameters.

  7. Once you’re done, select Save.

Add visibility logic for an item in the document

You can add visibility logic for an item in the document. An item can be the Table of Contents, a page break, a grouping area, a text area or a guidance.

To set visibility for an item:

  1. Select the item that you want to set visibility for.

  2. Select Settings () | Edit Visibility to display the Visibility Settings dialog.

    3. You can also select the Visibility () | Settings ().

  3. Select Hide or Show from the blue drop-down list to specify the visibility action.

  4. Select all or any to determine whether all conditions must be met or one condition must be met for the visibility action (from Step 3) to apply.

  5. Select Add Condition.

  6. Choose a condition type from the list, then specify the condition parameters.

    7. See Visibility Conditions for a detailed description of the available condition types and parameters.

  7. Once you’re done, select Save.

Add visibility logic for lists in text areas

You can add visibility logic for lists in text areas.

To set visibility for a list:

  1. Select Bullet List () or Ordered List () to insert a list in the text section.

  2. Select Insert Conditional Item () to add the visibility option to the list.

    3. The Visibility icon () displays next to the list item.

  3. Select the Visibility icon () to display the Visibility Settings dialog.

  4. Select Hide or Show from the blue drop-down list to specify the visibility action.

  5. Select all or any to determine whether all conditions must be met or one condition must be met for the visibility action (from Step 4) to apply.

  6. Select Add Condition.

  7. Choose a condition type from the list, then specify the condition parameters.

    8. See Visibility Conditions for a detailed description of the available condition types and parameters.

  8. Once you’re done, select Save.

You can select Clear to clear all conditions and start over.

You can also copy the visibility settings to another item. Select Copy to copy the visibility settings into clipboard, then open the Edit Visibility dialog for the other item and select Paste to paste the copied settings.

Visibility icons

Once you’ve set the visibility, one of the following eye icons displays next to a selected item:

Description

Show or hide an item regardless of visibility settings

You can make an item show or hide despite of the defined visibility settings or even if it doesn’t have any visibility settings. For example, if you want an item that’s hidden (due to visibility settings) to temporarily show until you finish authoring.

To show or hide an item:

  1. Select the visbility icon () or () for the item.

    The visibility options popup displays.

  2. Select Show or Hide.
