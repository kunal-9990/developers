Add a document reference in a checklist

Adding a document reference inserts a link to an existing document in the engagement file. For example, you can add a link to a query document.

Add a document reference in the Guidance tab

You can add a reference to a document in the Guidance tab.

To add a document reference in the Guidance tab:

  1. Select the Guidance tab.

  2. Select the text field to display the formatting toolbar.

  3. Select Insert Document Reference ().

    4. A list of the available documents in the engagement file displays. You can select Show hidden documents to select a hidden document if needed.

  4. Select the document you want to reference from the list.

    5. A reference to the selected document is created and added in the Guidance tab.

Add a document reference in the procedure description

You can add a reference to a document in the procedure description .

To add a document reference in the procedure description:

  1. Go to the procedure where you want to insert a document reference.

  2. Select the text field to display the formatting toolbar.

  3. Select Insert Document Reference ().

    4. A list of the available documents in the engagement file displays. You can select Show hidden documents to select a hidden document if needed.

  4. Select the document you want to reference from the list.

    5. A reference to the selected document is created and added in the procedure description.

Add a document reference in the guidance information for a procedure or a group

You can add a reference to a document in the guidance information for a group, a subgroup or a procedure.

To add a document reference in the guidance information for a procedure or a group:

  1. Go to the item where you want to add a document reference.

    2. The item can be a group, a subgroup or a procedure.

  2. Select the light-bulb icon () next to the item.

    3. The Guidance section opens in the left pane.

  3. Select the text field to display the formatting toolbar.

  4. Select Insert Document Reference ().

    5. A list of the available documents in the engagement file displays. You can select Show hidden documents to select a hidden document if needed.

  5. Select the document you want to reference from the list.

    6. A reference to the selected document is created and added in the guidance area.
