Carry forward an engagement

  • What is carried forward
  • What is not carried forward
  • Carry forward modified procedures, letters and responses
  • Carry forward new documents
  • Carry forward materiality

This feature is only available with products on the CaseWareCloud SE platform. Available SE products include OnPoint PCR — Preparation, Compilation and Reviews and Fusion RCT — Review, Compilation and Tax.

You can carry forward an engagement to re-use information, copying over answers from the prior year that have not changed.

The carry forward process uses answers from the prior year when making decisions again in the current year. For auditors and accountants, this is often a time-saving utility where they accept all of last year's changes or only work on the individual items that have changed.

What is carried forward

These are carried forward:

  • Trial balance values (imported data and adjustments)
  • Documents added by the engagement team in the prior year that were created in the engagement or uploaded directly
  • New checklist procedures
  • New text areas in letters, financials and memos
  • Changes to text areas in letters, financials and memos
  • New dynamic tables in financials
  • Changes to dynamic tables in financials
  • Modifications to checklist procedures
  • Responses to checklist procedures
  • Procedures that were modified in a previous year
  • Issues of Consideration for next year type
  • Materiality statements

If the primary author has made any changes to an item, end-users will not be able to carry it forward. For example, end-users cannot carry forward a procedure that has been deleted by the primary author. Also, if a word is changed in a procedure’s template, end-users cannot carry forward any changes to that word.

What is not carried forward

These are not carried forward:

  • Annotations
  • Issues that are not Consideration for next year type
  • Control Report
  • Risk Report and risks
  • Risk Assessment Report
  • Queries
  • Content that was modified by the end-user if the same content was modified by the primary author

To carry forward an engagement:

  1. Select Create Next Period's File from the More actions [...] icon.

  2. Complete the Carry Forward dialog as required.

  3. Select Save.

    The new engagement is created.

Carry forward modified procedures, letters and responses

You can carry forward a procedure, letter or financial statement section that was modified in a previous period, as well as responses.

To carry forward a procedure or letter that was modified in a previous period:

  1. Open an engagement file that was previously carried forward.

  2. From the Documents tab, open a procedure that was modified during the previous engagement.

  3. Accept () or Ignore () changes as required.

  4. Use the Carry forward icon ()to display bulk carry forward choices.

 

You can also carry forward responses to procedures. You must accept or dismiss any procedure modifications before carrying forward responses.

To carry forward a response from a previous period:

A response cannot be carried forward if the procedure that is related to it is not carried forward first. This only applies if the procedure has carry forward options.

  1. Open an engagement file that was previously carried forward.

  2. From the Documents tab, open a procedure whose response was modified during the previous engagement.

  3. Accept () or Ignore () changes as required.

  4. Use the Carry forward icon () to display bulk carry forward choices.

Carry forward new documents

You can carry forward documents that were added to an engagement in a prior period.

To carry forward a document that was added in the previous year:

  1. Open an engagement file that was previously carried forward.

  2. Select the Add document icon () to carry forward the new document.

  3. Select the Ignore suggestion icon () to prevent carrying forward a specific document.

  4. Use the Carry forward icon () to display bulk carry forward choices.

Carry forward materiality

You can carry forward values in the materiality statement from a previous engagement.

To carry forward materiality from a previous engagement:

  1. Open an engagement file that was carried forward that includes entries in the materiality statement.

  2. From the Documents tab, select the materiality statement.

  3. Use the Carry forward icon () to display bulk carry forward choices.
