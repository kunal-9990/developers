Upload documents to a workspace

You can add files and documents to Cloud so that all of the relevant material for your engagement is stored in one place, well organized, and easy to find and access.

Upload a file

If you have any files that will be used as a reference during the engagement, you may want to upload them to Cloud. These files might include copies of correspondence with the client or documents related to engagement work.

To upload a file:

  1. Ensure that you have the Editor role or the equivalent privileges.

  2. From the Cloud menu (), select Files.

  3. Select the appropriate entity using the Entity Selector.

  4. Select New, and then choose Upload. The File Upload dialog is displayed.

  5. Use the Select Files button, or drag and drop a file to upload it.

  6. In the Share Settings dialog, choose sharing options for the file. Select OK.

Upload a file larger than 1GB

Files larger than 1GB must be uploaded with the Advanced Uploader. Using the Advanced Uploader requires Java and Microsoft Internet Explorer.

To upload a file larger than 1GB:

  1. Ensure that you have the Editor role or the equivalent privileges.

  2. Ensure you have installed the latest version of Java, and are using a browser that supports it.

  3. From the Cloud menu (), select Files.

  4. Select the appropriate entity using the Entity Selector.

  5. Select New, and then choose Upload. The File Upload dialog is displayed.

  6. From the File Upload dialog, select Use advanced uploader.

  7. Select the file you want upload, then select Add.

  8. In the Share Settings dialog, choose sharing options for the file. Select OK.

Your file is uploaded. For troubleshooting information, see Java applet is not loading.

Link to a Google doc

If your team uses Google Drive to collaborate on your engagement work, you link to your Google files in Cloud. Before You'll need to link your Google account to your Cloud profile to get started - for more information, see Integrate productivity tools with Cloud.

To link a Google doc:

  1. Ensure that you have the Editor role or the equivalent privileges.

  2. From the Cloud menu (), select Files.

  3. Select the appropriate entity using the Entity Selector.

  4. Select New, then choose Google Drive | Existing Google File.

  5. In the Reference dialog, paste the sharing link for the Google file and enter a description for the file.

    For more information about generating a shareable link to a Google document, see Google's support.

  6. Select Save.
