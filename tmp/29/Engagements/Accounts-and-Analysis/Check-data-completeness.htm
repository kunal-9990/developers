Check data completeness

This feature is only available with CaseWare Analytics.AI, part of the CaseWare Cloud suite.

Learn more about Cloud.

You can use a data completeness check to see whether the account balances calculated from the trial balance match the account balances calculated from the transactions. Discrepancies may indicate that you have received or uploaded an incomplete data set and may need to re-import the data.

Access Data Completeness from the Documents page.
