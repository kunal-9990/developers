Add an account or group to the trial balance

This feature is only available with products on the CaseWareCloud SE platform. Available SE products include OnPoint PCR and CaseWare ReviewCompTax.

When you create your engagement, standard groupings are automatically populated in the trial balance. If you require a custom group or account, you can manually add them to the trial balance.

Add a group

If you require a custom group not listed in the existing trial balance, you can add it on the Trial Balance page. New groups require a name and parent group before you can assign accounts.

To add a group:

  1. From the top menu, select Trial Balance.

  2. Select the grouping menu and choose one of the available grouping structures.

    The available groupings are:

    • Financial
    • Tax Code
    • Chart of Accounts
    • GIFI (only available for CaseWare RCT)

  3. Select theAdd () | Group.

  4. Enter a name for your group, and select a parent group, if applicable.

  5. Select Save.

The new group is added to the trial balance. You can find it listed in the trial balance under the parent group and assign any relevant accounts.

Add an account

You can enter new financial data in your trial balance by creating new accounts. New accounts require a unique number and name before you can enter values.

To add an account:

  1. From the top menu, select Trial Balance.

  2. Select the grouping menu and choose one of the available grouping structures.

    The available groupings:

    • Financial
    • Tax Code
    • Chart of Accounts
    • GIFI (only available for CaseWare RCT)

  3. Select the Add () | Account.

  4. In the Account dialog, enter a number and name for your account.

  5. Select Save.

The new account is added to the trial balance. The Assign Accounts icon () on the trial balance toolbar now indicates that there is an unassigned account that needs to be assigned to a group.
