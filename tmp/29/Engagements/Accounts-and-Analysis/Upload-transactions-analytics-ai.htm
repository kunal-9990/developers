Upload transactions in Analytics.AI

This feature is only available with CaseWare Analytics.AI, part of the CaseWare Cloud suite.

Learn more about Cloud.

You must upload your transactions for a risk analysis to run.

If you uploaded your trial balance from Xero or QuickBooks online, and included transactions, you can skip this procedure.

To upload your transactions:

  1. Select 0-4 Upload Transactions from the Documents tab.
    The Import Transactions page displays.

  2. Drag and drop the transactions file, or click and browse to the file as indicated.
    The Import Transactions page is populated, similar to the following:

  3. Use the pull-down menus to map each data type to a specific column.

  4. Select the Import button () to import your mapped transactions.
