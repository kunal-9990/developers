To populate the trial balance, you’ll have to import the financial data to your engagement file. In some cases, you can import this data directly from another accounting software package using the native import utility (for online accounting software) or a desktop utility import we provide.

Importing from online accounting software

You can import your trial balance directly from online accounting software. You can currently import from the following software products:

  • Xero

  • QuickBooks Online

  • Exact Online

To import client data from Xero:

  1. Ensure that you have the Editor role or equivalent privileges. To connect to the Xero file, ensure that you have a valid Xero account and that you've been assigned the Standard user or Auditor role on the file. For more on users and roles in Xero Online, see their help materials.

  2. From the top menu, select Trial Balance.

  3. On the trial balance page, choose a trial balance grouping structure from the drop-down menu at the top of the page.

  4. Under Import from the following platforms, select Xero.

  5. In the new dialog, select Login.

    A new window is displayed prompting you to log in to your Xero account.

  6. Enter your Xero account information and select Login.

  7. Select your client organization in Xero and select Allow access for 30 mins.

  8. In your SE engagement, check the box beside the correct client file. If your client file includes transactional data, you can import that data by selecting the option.

  9. Select Import.

To import client data from QuickBooks Online:

  1. From the top menu, select Trial Balance.

  2. On the trial balance page, choose a trial balance grouping structure from the drop-down menu at the top of the page.

  3. Under Import from the following platforms, select Quickbooks.

  4. In the new dialog, select Login.

    A new window is displayed prompting you to log in to your QuickBooks Online account.

  5. Enter your QuickBooks Online account information and select Sign In.

  6. Select your client organization in QuickBooks Online.

  7. Select Authorize to share your data between CaseWare Cloud Import and Intuit.

  8. In your SE engagement, check the box beside the correct client file. If your client file includes transactional data, you can import that data by selecting the option.

  9. Select Import.

To import client data from Exact Online:

  1. From the top menu, select Trial Balance.

  2. On the trial balance page, choose a trial balance grouping structure from the drop-down menu at the top of the page.

  3. Under Import from the following platforms, select Exact.

  4. In the new dialog, select Login.

    A new window is displayed prompting you to log in to your Exact Online account.

  5. Enter your Exact Online account information and select Login.

  6. In your SE engagement, check the box beside the correct client file. If your client file includes transactional data, you can import that data by selecting the option.

  7. Select Import.

When you import your trial balance from online accounting software, the accounts and groupings are automatically mapped.

Importing from CaseWare Working Papers

Note: CloudBridge integration is only supported by Fusion RCT — Review, Compilation and Tax.

To import client data from a Working Papers file, you'll need to use CloudBridge, an add-on for Working Papers that transfers the data from your Working Papers engagement file to a Cloud engagement.

Note: This procedure requires a Working Papers engagement file. Your copy of Working Papers must also be integrated with your Cloud firm. For more about syncing Working Papers to Cloud, see our Working Papers help.

Download the CloudBridge installation package below.

CloudBridge (.cwp)

 

To create a link to CloudBridge in Working Papers

  1. Open the CloudBridge installation package and follow the steps in the installation wizard.

  2. Open your client file in CaseWare Working Papers.

  3. From the Document tab, click From Library | CloudBridge.

  4. The Document Library dialog displays. Select CLBG and click Add.

The CLBG (CloudBridge) document displays at the top of the Document Manager.

Importing from third-party desktop accounting software

To format client data from third-party desktop accounting software, use one of our free import tools. The tool you need depends the accounting software you're using.

CaseWare Transformer

The following desktop accounting software packages require CaseWare Transformer:

  • AuditXML 3.x (Beta)
  • MyOB AccountRight Live 2013-2016
  • Pastel Partner 12.1 - 14.0 (Beta)
  • QuickBooks CA 2012 - 2017
  • QuickBooks US 2012 - 2017
  • Sage 50 Accounting (Peachtree-WIN) 2012 - 2017

Download CaseWare Transformer: 

CaseWare Transformer (.exe)

 

Client data from all other desktop accounting software should be imported using the CaseWare Cloud Import Utility.

CaseWare Cloud Import Utility

The CaseWare Cloud Import Utility supports most desktop accounting software. For a complete list, see Supported third-party desktop accounting software.

CaseWare Cloud Import Utility - Beta (.exe)

 

To format client data using the Cloud Import Utility:

  1. Download the utility.

  2. Open the Utility. Select Other Data Source, then select Next.

  3. In the Import From and Version fields, specify the accounting software you're exporting from.

  4. Browse for the Import Path of the client file you want to import. If multiple client files are located in this folder, you will need to specify which file you want to import in the Select Client ID File field.

  5. Select Next. You will see a prompt with the output folder you'll need to open to access your new file. Select Finish to perform the conversion.

The generated output file can be uploaded into a variety of CaseWare Cloud products.

Importing from Excel/CSV

If the accounting software package you have is not supported by any of our import utilities, you can export your client data to a CSV (comma-separated value) file and then perform the import using this file. The format of the file can be either Excel or .csv.

To import client data from Excel/CSV:

  1. From the top menu, select Trial Balance.

  2. On the trial balance page, choose a trial balance grouping structure from the drop-down menu at the top of the page.

  3. Under Import from, select Excel or CSV. In the upload dialog, drag and drop an Excel file or a CSV file into the gray upload area, browse for the file on your local device storage or select the file from the engagement if you have already uploaded it.

  4. Map the columns in the import file by choosing the field that best matches the data from the Import as drop-down menu. If you don’t want to import a column, select Do not import to exclude it.

  5. Select Import.

Your imported client data populates the trial balance.

Troubleshooting

Problem

You follow all the steps to import data from online accounting software, but the process finishes and no data is imported.

Solution

If the year end date in the engagement and the source data file do not match, the trial balance is not imported. Open the engagement properties and ensure that the year end date is correct.
