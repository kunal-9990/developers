You can carry forward an engagement to re-use information, copying over answers from the prior year that have not changed.

The carry forward process uses answers from the prior year when making decisions again in the current year. For auditors and accountants, this is often a time-saving utility where they accept all of last year's changes or only work on the individual items that have changed.

What is carried forward

These are carried forward:

  • Trial balance values (imported data and adjustments)
  • Documents added by the engagement team in the prior year that were created in the engagement or uploaded directly
  • New checklist procedures
  • Modifications to checklist procedures
  • Responses to checklist procedures
  • Procedures that were modified in a previous year
  • Issues of Consideration for next year type

If the primary author has made any changes to an item, the end-user will not be able to carry it forward. For example, end-users cannot carry forward a procedure that has been deleted by the primary author. Also, if the primary author changes a word in a procedure, the end-user cannot carry forward any changes to that word.

What is not carried forward

These are not carried forward:

  • Annotations
  • Issues that are not Consideration for next year type
  • Control Report
  • Risk Report
  • Risk Assessment Report
  • Queries

To carry forward an engagement:

  1. Select Create Next Period's File from the More actions [...] icon.

  2. Complete the Carry Forward dialog as required.

  3. Select Save.

    The new engagement is created.

Carry forward modified procedures and responses

You can carry forward a procedure that was modified in a previous year, as well as responses to procedures.

To carry forward a procedure that was modified in a previous year:

  1. Open an engagement file that was previously carried forward.

  2. From the Documents tab, open a procedure that was modified during the previous engagement.

  3. Use the Accept () and Ignore () icons as required for the individual changes.

  4. Use the Carry forward icon ()to display bulk carry forward choices.

 

You can also carry forward responses to procedures. You must accept or dismiss any procedure modifications before carrying forward responses.

To carry forward a response from a previous year:

  1. Open an engagement file that was previously carried forward.

  2. From the Documents tab, open a procedure whose response was modified during the previous engagement.

  3. Use the Accept () and Ignore () icons as required for the individual changes.

  4. Use the Carry forward icon () to display bulk carry forward choices.

Carry forward new documents

You can carry forward documents that were added to an engagement in a prior year.

To carry forward a document that was added in the previous year:

  1. Open an engagement file that was previously carried forward.

  2. Select the Add document icon () to carry forward the new document.

  3. Select the Ignore suggestion icon () to prevent carrying forward a specific item.

  4. Use the Carry forward icon () to display bulk carry forward choices.
