A firm author is a specialist that uses a template to customize content to meet the firm’s needs

Firm authors can follow these instructions to customize product content.

Customize a template

Use a template to make limited customizations to your product.

To customize a template:

  1. From the Cloud menu, select Settings.

  2. Select Manage Template from below your product.

  3. Select Create Draft. A draft template is created, and opens in a new tab.

  4. From the template, select Information () | Settings.

    The Product Settings dialog displays. Its contents vary depending on your product.

  5. Change the default Currency, Dates, Numbers and Fonts settings as required, then select Close.

    The Documents tab displays.

  6. Select More actions () | Hidden Items to display other available documents.

  7. Select a document to open it, then select Toggle edit view ().

  8. Edit the document as desired, then select Toggle edit view ().

    Turning questions off in checklists is not recommended, as some questions drive the logic behind document visibility.

  9. Return to the Cloud menu. Select Settings, then Manage Template from below your product.

  10. Select Delete Draft () to delete the changes you just made.

    Or

    Select Publish Template (). The number is your template version.

New engagements will have the customizations included that you added to the template.

Update template with product changes

On occasion, changes may be made to your product’s master template that you want to incorporate into your product. You can do this by updating your firm template. The procedure varies slightly depending on whether or not you have an existing draft.

To update the template with product changes when there is no existing draft:

  1. From the Cloud menu, select Settings.

  2. Select Manage Template from below your product.

  3. Select Update. A draft template is created.

  4. Select Open Draft.

    Verify the changes to the product.

  5. Return to the Cloud menu. Select Settings, then Manage Template from below your product.

  6. Select Publish Template (). The number is your template version.

New engagements will have the customizations that were added to the product.

To update the template with product changes when there is a existing draft

  1. From the Cloud menu, select Settings.

  2. Select Manage Template from below your product.

  3. Publish or delete the current draft.

  4. Select Update. A draft template is created.

  5. Select Open Draft.

    Verify the changes to the product.

  6. Return to the Cloud menu. Select Settings, then Manage Template from below your product.

  7. Select Publish Template (). The number is your template version.

New engagements will have the customizations that were added to the product.
