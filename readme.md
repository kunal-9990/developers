## Deployment

composer install
npm install
npm run production

update the .env variable PATH_TO_PUBLIC

## Adding a help version

Help content is organized by product and then version within /public/documentation_files. Each version directory must contain:

- csh_redirect.xml
- Online Epics.xml
- OnlineOutput.xml
