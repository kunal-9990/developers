Installing CorpTax

In this guide, you will learn:

  • Integrating Working Papers file with CaseWare Cloud
  • Publishing File to Cloud Server
  • Installing the Tax Bridge
  • Creating a link for the Tax Bridge
  • Using Tax Bridge

Integrating Working Papers file with Cloud

  1. Open the applicable Working Papers file.
  2. On the Tools tab, in the Options group, click Options.

  3. Click Options | CaseWare Cloud and select Integrate with CaseWare Cloud.
  4. Type your URL in the Server field. Do not include the web protocol (http:// or https://). Example: us.cwcasewarecloud.com/docfirm

  1. Click OK.
  2. Provide your Cloud credentials on the CaseWare Cloud Login dialog.

Publishing File to Cloud Server

  1. On the Cloud tab, click Publish.

  2. Select the appropriate server where the file is to be published. If there are no servers, from the Server drop-down menu, select Add Server and enter the server location and label.

  3. (If applicable) Select the appropriate folder from the Location drop-down menu.

  4. Click OK.

Installing the Tax Bridge

  1. Run the .cwp file and follow the instructions of the CaseWare Template Installer Wizard.

  2. Before actually installing a template, the End User License Agreement (EULA) must be accepted. To continue with the installation, click I accept the terms in the license agreement and click Next.

  3. Specify the location where the tax bridge should be installed. Default paths are automatically displayed based on the location of the Working Papers installation. Click Next.

  4. Click Close.

Creating a link for the Tax Bridge

  1. Return to the Document Manager, right click and select New| Document Link.

  2. In the New Document Link dialog, enter TaxBridge as the Name and in the File/URL field enter the following address: cw:program/scripts/taxbridge/index.html

  3. Click OK to add the document link.
  4. Double-click to open the link.
