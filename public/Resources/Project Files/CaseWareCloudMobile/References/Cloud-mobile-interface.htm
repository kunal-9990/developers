Functionality differences in the mobile interface

Certain features, while available on mobile devices, function differently than their desktop or tablet counterparts. To access most features on mobile, you will need to select the Add ()button.

Activities

To add a new Discussion, Task, or File Request, select Add.

Files

To upload files from your mobile device, or create a new file using Cloud select Add | Files.

Entities

To create a new entity, select Add | Entities.

Other Engagement

To create a new Other Engagement, select Add | Files | Other Engagement.

Time

  • To add an entry for yourself, select Add | Time.

  • When accessing the Time app on a mobile device, you cannot add or edit time entries for other users.

  • You are restricted to the Basic view.

Expense

  • To add an entry for yourself, select Add | Expense.

  • When accessing the Expense app on a mobile device, you cannot add or edit expense entries for other users.

  • You are restricted to the Basic view.

Staff

  • To add a staff member, select Add | Staff.
  • You cannot manage or export views.
  • You cannot import new staff members from a CSV.

Contacts

  • To add a contact, select Add | Contact.
  • You cannot manage or export views.

Groups

You cannot create new groups.
