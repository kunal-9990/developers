Gathering client info

You have completed all of the steps in the engagement acceptance stage:

  • completed the Optimizer checklist

  • reviewed and finalized the draft version of the engagement letter

  • sent a copy of the engagement letter to the client to be signed

  • added the signed engagement letter to the engagement

Now you will want to import the client’s trial balance to proceed with your engagement.

Complete the Client inquiries document

Open the Documents page, and in the Planning section, open 1-60 Client queries- Request for compilation documents and records. This query document includes a list of file requests for files and information that could be relevant for a review engagement.

Modifying the document

Since there may be documents not relevant for this client or engagement, you can edit the query to make it more manageable for both you and the client. At the top-right of the document, select the Edit icon ().

All of the sections in this query document are now available for editing. You can select a text area to update the request description, you can use the Move option to move a procedure to a different place in the document, or you can use the Delete icon () to remove individual procedures or entire sections.

Note: When you delete a procedure or a section in a query, a confirmation dialog is displayed warning you that deleted procedures cannot be reinstated. If you remove content from a query by mistake, you can add a new copy of the original query to your engagement file on the Documents page.

Request documents

Change the status to Open to display the Open Query dialog.

Choose a due date, and enter the following custom email message:

Please attach the Trial balance, data file, and entity documents for this year's compilation.

Assign the task to all contacts at the client firm or select an individual user, then select OK to open the query.

Once the client attaches their files, you can review the documents before accepting the procedure. The attached documents can be opened by selecting them.

Note: When you attempt to open an Office file, you will be prompted to confirm that you want to open the file in Office desktop.

Once you have confirmed that client’s file contains the trial balance information that you need, you can accept the procedures and complete the query. Select the Responded icon beside the Chart of accounts procedure and choose Accept.

Import the trial balance data

Now that you have added the client’s trial balance file to the engagement file, you can import the trial balance information.

Compilation and Review -- Canada offers multiple options for performing the import of the trial balance's data.

CaseWare Transformer Import

If you have client data formatted in one of the following software packages, you can use the free CaseWare Transformer tool to import the data into your engagement:

  • AuditXML 3.x
  • MyOB AccountRight Live 2013 - 2016
  • Pastel Partner 12.1-14.0
  • QuickBooks CA 2012 - 2017
  • QuickBooks US 2012 - 2017
  • Sage 50 Accounting (Peachtree-WIN) 2012 - 2017

For more on creating using the tool to create Transformer files, please see our online help.

To import a Transformer file:

  1. From the top menu, select Trial Balance.

  2. On the trial balance page, choose a trial balance grouping structure from the drop-down menu at the top of the page.

  3. Under Import from, select Desktop Accounting Software. The upload dialog is displayed.

  4. If your file includes transactional data, check the box beside Import available transactional data to include that data in the import.

  5. Drag and drop a transformed zip file into the gray upload area, browse for the file on your local device storage, select the file from the engagement if you have already uploaded it, or import the file from CaseWare Cloud.

Online Imports

If you have credentials for Exact Online, QuickBooks Online, or Xero Online, you can import the trial balance information from an existing file by granting Compilation and Review access to the online file's data.

To import client data from Exact Online:

  1. From the top menu, select Trial Balance.

  2. On the trial balance page, choose a trial balance grouping structure from the drop-down menu at the top of the page.

  3. Under Import from the following platforms, select Exact.

  4. In the new dialog, select Login.

    A new window is displayed prompting you to log in to your Exact Online account.

  5. Enter your Exact Online account information and select Login.

  6. In your SE engagement, check the box beside the correct client file. If your client file includes transactional data, you can import that data by selecting the option.

  7. Select Import.

To import client data from QuickBooks Online:

  1. From the top menu, select Trial Balance.

  2. On the trial balance page, choose a trial balance grouping structure from the drop-down menu at the top of the page.

  3. Under Import from the following platforms, select Quickbooks.

  4. In the new dialog, select Login.

    A new window is displayed prompting you to log in to your QuickBooks Online account.

  5. Enter your QuickBooks Online account information and select Sign In.

  6. Select your client organization in QuickBooks Online.

  7. Select Authorize to share your data between CaseWare Cloud Import and Intuit.

  8. In your SE engagement, check the box beside the correct client file. If your client file includes transactional data, you can import that data by selecting the option.

  9. Select Import.

To import client data from Xero:

  1. Ensure that you have the Editor role or equivalent privileges. To connect to the Xero file, ensure that you have a valid Xero account and that you've been assigned the Standard user or Auditor role on the file. For more on users and roles in Xero Online, see their help materials.

  2. From the top menu, select Trial Balance.

  3. On the trial balance page, choose a trial balance grouping structure from the drop-down menu at the top of the page.

  4. Under Import from the following platforms, select Xero.

  5. In the new dialog, select Login.

    A new window is displayed prompting you to log in to your Xero account.

  6. Enter your Xero account information and select Login.

  7. Select your client organization in Xero and select Allow access for 30 mins.

  8. In your SE engagement, check the box beside the correct client file. If your client file includes transactional data, you can import that data by selecting the option.

  9. Select Import.

CSV file (Excel) Import

Select Browse. A new dialog is displayed prompting you to select a .csv (Excel) file from your computer.

Select the file from this dialog, and the Column Mapping dialog is displayed next.

For each column in the import file, select a field to import the values in that column to.

At the bottom of the dialog, select the Import button.

The contents of the trial balance file are now imported into the engagement file. The net income is displayed at the top corner of the window, and if the trial balance is out of balance, warnings are displayed at the top of the relevant columns.

Modify the trial balance

When you import a trial balance, Review and Compilation – Canada will map as many of the accounts (and assign the appropriate GIFI codes, which will also be automatically transferred to the tax forms if you're performing a tax return) as it can, but in some cases, you may need to evaluate accounts and assign them manually. In this example, notice that the Assign accounts button now has a number in parentheses.

This indicates that one account is not currently assigned to a group. Select the button to open the Assign accounts page.

On this page, one account, Cash, is displayed in the Unassigned column.

Select an account to move it into the Selected Accounts section. When you do, Review and Compilation – Canada displays a few group suggestions based on the account name, “Cash”. To assign it to the Cash group, select the green Assign icon that is displayed beside the suggestion.

You can also drag and drop accounts on this page to assign them to a new group. In this example, in the Groups column, the account Bank - Savings is currently grouped under Cash and Cash Equivalents, but Bank - Checking is grouped one level lower, under Cash. Select Bank - Savings and drag it one line down, to the Cash group. When the group line is displayed with a green background, you can drop the account to assign it to the new group.

Return to the trial balance page. You can see that your changes are reflected here, as there are now three accounts in the Cash group.

Annotate the trial balance

While you are reviewing the trial balance, you can annotate accounts. From the trial balance menu, select the View button.

You can select different options here to change the information displayed on the trial balance. First, select Variance. You will see a new column that displays the variance between the final balance in the current period and the final balance from the prior period.

Notice that in this example, there is no Prior value for the Bank - Savings account. You ask the client about this, and they inform you that the cash in the savings account came from sales that happened very late at the end of the year. You can add an annotation to include this information directly on the trial balance.

Select the Add annotation button () on the Bank - Savings line. A new dialog is displayed.

When you select this option, you can choose a purpose for your annotation, link it to a document or to an adjustment, and add a note to describe the reason for the annotation. You may also see a separate area where you can choose from several annotations suggested by Review and Compilation – Canada.

For this annotation, you select Variance for the purpose, and for the note section enter “Late sales at end of year.” and save your annotation.

Now the trial balance displays an icon for the annotation type, and it includes the description from the note.
