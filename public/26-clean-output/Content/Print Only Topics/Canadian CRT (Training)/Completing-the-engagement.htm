Completing the engagement

With your compilation work complete, you are ready to start preparing your conclusions and verifying that the engagement team has complied with government reporting standards.

Complete the conclusion checklists

The Forming a conclusion section lists the final documents you'll need to complete to wrap up the engagement work. Once you've completed all procedures and solicited all relevant client communications in this section, your engagement work will be complete.

Finalizing the financial statements

At the conclusion of the engagement, you will create a financial statements document for the client where you explain your findings and present a conclusion. Review and Compilation – Canada has a built-in template that will make it easier for you to prepare the financial statements.

On the Documents page, in the Issuing the report section, open the 6-10 Report transmittal document. In this document, you'll specify the distribution methods for the report, authorize it for release, and set milestone dates such as the release and final assembly dates.

Open 6-20 Financial statements.

The Financial statements document includes the following sections:

  • Title page

  • Table of Contents

  • Report section

  • Financial statement tables

    • Balance Sheet

    • Statement of Income

    • Statement of Retained Earnings

    • Statement of Cash Flows (if applicable)

  • Notes to the Financial Statements (if applicable)

Customize the financial statements

Most of the content in the financial statements document is already populated for you. The content in the review report is based on your answers to the questions in the engagement planning documents. Groups and accounts on the financial statements tables are taken from the trial balance.

However, you will need to make other changes to customize the financial statements for your client.

As you saw in the engagement letter, the financial statements include pulldown menus. These must be used to complete the statement in some sections.

Once you have made all changes that you need to, select the Edit icon again to save your changes.

Edit a financial statement table

While you cannot change the specific accounts and groups that are included in a financial statement table, you can make some changes to the content that is displayed.

Enter information into a financial statement table

Many of the tables in the financial statements are populated automatically. The information in them is pulled straight from the trial balance, to make financial statement preparation easier for you. However, you will also likely need to enter some additional information yourself.

Some of the lines in the financial statements are input rows. In an input row, you can enter amounts into the fields, and these amounts will be included in final calculations for the table.

For example, select Edit and scroll down to the Statement of Cash Flows.

Normally, rows with no values are hidden from the standard view of a table, but in edit mode you can see all of the input rows, such as the Other non-cash income (expense) row. Once you enter any financial data into an input row, that row will be displayed in the statement of cash flows.

Note: If you want to enter a different amount in a row that already has automatic values, select the gear icon () at the end of the row and then select Override calculated balances.

Expand a line in the financial statements

Many of the financial statement tables collapse accounts into their groupings by default. You can expand lines in the financial statement to give more details about some areas of the engagement file.

For example, in the Balance Sheet, the Long-Term Assets section displays totals for two areas: Property, plant and equipment and Other long-term assets. You decide that you want to give the client more information about the Property, plant and equipment group.

Beside the group name, there is an Expand lines icon(). Select this icon.

All of the lines under your selected group are now displayed, and you can see the amounts for each different category under Property, plant and equipment - Cost, along with the amount for Accumulated amortization of property, plant and equipment..

You then decide that you do not need to show the client all of the subcategories under Property, plant and equipment. Select the Collapse lines icon () beside this group.

You have now collapsed the subgroups.

Note: If you want to see more details about a balance on the financial statement without changing a table, you can select the line with that balance to drill down and display more details.

Modify the description of a financial statement line

To change the descriptions of lines in the financial statements, you will need to change the document settings. At the top of the page, select Edit. Select the Settings button, then enable the option that says “Allow override of line”.

Once you have selected the Done button to save this change, you will be able to edit the description for any line item in the document.

Notes to the financial statements

The Notes to the Financial Statements section is at the bottom of the financial statements document. If you need to provide any supplementary information about the results of your engagement, you can add that information here.

Each note can include a guidance section with information on any changes that you might need to make to it as the preparer. They also typically have one or more text areas that contain explanations relevant for the note.

Some of the notes in your engagement may also include a dynamic table. Dynamic tables contain specific lines from the trial balance or custom rows where you can input your own data. These tables are intended to highlight specific parts of the trial balance or to support your conclusions.

If required, you can also amend this section by adding notes.

Add a cross reference to a line

If a line in the financial statement table requires more explanation, you can add a cross reference to the line that links to one of the notes to the financial statements.

For example, you decide that you want to provide a breakdown of the Property, Plant and Equipment group, but you do not want to display it on the balance sheet. You can link a line in the balance sheet to the Property, plant and Equipment note.

At the top of the balance sheet, select Edit.

When you enter edit mode, you will see that each group name now has a # link in the Note column. Select the # link for the Property, Plant and Equipment, net group.

A pop-up is displayed, and you can see the name of the note to the financial statements that you created. Select the checkbox beside 6. Property, plant and equipment..

Select the Edit icon again to save your changes. The Property, Plant and Equipment group now includes a reference to the note in the Note column.

Print the financial statements

Once you have finished preparing the client’s financial statements and your reviewer has signed off on the document, you will want to print a copy of the financial statements to present to the client. You might also print other copies of the document while you are still in the process of working on it in order to review the impact your changes will have on the finished product.

You cannot print directly from Review and Compilation – Canada. Instead, printing creates a PDF copy of the file for your review. In the financial statements menu at the top of the page, there are two commands for printing: Print and Print Settings.

Select the Print Settings button to display the Print Settings dialog. You can enter content for the header and footer of your document, and you can also set the page size and orientation, the margin width, and the table of contents settings. If you make any changes, select Save to close the dialog.

Select Print to generate a PDF copy of the Financial Statements. This automatically opens a new tab in your browser window, and the PDF version of the trial balance is displayed. You can choose to save the file as a PDF to your local hard drive, or you can select a local printer to print a physical copy.
