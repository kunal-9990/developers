Performing fieldwork

You have performed all of the steps involved in adding the trial balance information to your engagement file:

  • submitted and accepted a query for the client’s trial balance file

  • imported the trial balance into the engagement file

  • assigned an unassigned account and corrected the mapping

  • added an annotation to the trial balance

You are now ready to perform fieldwork, the main part of the review engagement.

Working on checklists

On the Documents page there is a series of checklist documents under the Performing section. These checklists are leadsheet documents, and you will use them to guide your analysis and document the results of your inquiries for the relevant financial statement areas.

Leadsheet checklists include supplemental sections that will help you to complete them more easily and quickly:

  • Analysis

  • Objective and Notes

  • linked query responses

Open the 3-A Cash document.

Analysis

At the top of each leadsheet document you will see an analysis section. This area pulls information from the trial balance and displays it in graphs or tables.

The analysis section presents data visually, and it only includes trial balance information for relevant accounts and groups, so it makes it easier to identify what inquiries you might need to make to your client as part of the engagement work.

You can view either preliminary or final balances using the drop-down menu at the top of the section, and you can choose which graphs and tables to show by switching the display toggles. Each module also has a section where you can enter comments about the data displayed there.

Objective and Notes

Each leadsheet document also includes an Objective and Notes section.

Just as you saw in the engagement letter draft, this section contains information about the current document and guidelines on how to complete it.

Adding the client contacts

In the Inquiries made of entity personnel section contained at the top of every leadsheet, you can enter the names of the contacts you corresponded with to complete the inquiries, and the dates of correspondence.

Linked query responses

In the Procedures section of each leadsheet document, you see a list of questions to be answered or fields to fill in. If you are using the 2-03 Client inquiries - Questions query document to gather information from your client, you may see linked query responses under some procedures.

These linked query response areas are linked to matching procedures in the PBC Requests - Review Inquiries query document. When you accept a response from a client in this query, the linked query response is displayed under the matching question on the leadsheet document.

Gathering more information from client

You are working on the Cash and Cash Equivalents document, and you reach a procedure that asks “Has there been a proper cutoff of receivables/cash receipts and disbursements?”

In the analysis section, you noticed that the cash total for this year is much higher than the totals from the prior years. You decide to contact the customer to ask them to verify the response to this question, since a late cutoff could have inflated the current year total.

Go to the Documents page and open the PBC Requests - Review Inquiries document. Scroll to the matching procedure. Select the Comment link under the procedure and enter some contextual information about what you want the client to confirm.

Note: The Review Inquiries query includes linked queries for many different leadsheet documents. In normal engagement work, you should identify all the procedures where you need client input and delete the rest of the procedures from the query before submitting it to the client.

Open the query. The client can enter information for the specific procedure and mark it as completed. You will see this response in the PBC Requests - Review Inquiries document, and if you are satisfied with the information they have provided then you can sign off on it. Select the Responded icon, and choose Accept.

Once you have accepted the procedure in the query, you can see the client's response in the linked procedure on the leadsheet document.

You can copy the client’s response to the query in the client response field, and if you are satisfied with the information they have provided then you can respond to the procedure and sign off on it.

Work on Microsoft Word or Excel files

If a member of the engagement team has attached files from Microsoft Word or Microsoft Excel to your engagement, you can open these files to review the contents or make changes to them. On the Documents page, select a file to open it.

When you enable editing in the Word or Excel file on your desktop, the file in your engagement is locked. Other members of the engagement team can open a read-only version of the file, but they will not be able to edit it until you have closed the file.

An icon is displayed on the Documents page to let other members of the engagement team know when a file is locked.

Add annotations to PDF documents

You can open PDF documents that have been attached to the engagement by selecting them on the Documents page. While working on the engagement, you can add annotations to a PDF document to note work that you have performed or to identify problems or potential inaccuracies in the document.

In the PDF file, select an area of the document to display the Add Annotation icon (). If you select a line of text, the icon is displayed at the end of the line. Otherwise, it is displayed at the point you select.

Select the icon to add an annotation at the indicated spot. You will choose an annotation type, link the annotation to a document or adjustment (if applicable) and enter a description for the annotation.

Once you select Save, the annotation is added to the PDF. An icon indicates what type of annotation it is, and other engagement team members can view the annotation by hovering their cursor over it or by selecting it.

Complete and get signoff on the remaining checklists and procedures in the Performing documents to complete the fieldwork for the engagement.
