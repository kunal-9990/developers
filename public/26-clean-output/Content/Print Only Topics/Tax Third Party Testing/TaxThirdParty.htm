Introduction

This guide contains instructions on exporting data from a TaxPrep file, sanitizing the data to remove proprietary information, and sending the sanitized data to CaseWare for testing purposes.

Requirements

You will need to have Corporate TaxPrep 2017 v1.1 installed on your system.

You will also need the SanitizeCSV.zip file package and the comprehensive.xuf filter file. Both can be found in the email you received from your point of contact at CaseWare.

Setup

Before you can export your data from TaxPrep. You’ll need to install the Xpress filter using the .xuf file contained in the email.

Installing the Filter

  1. If TaxPrep is currently open, close it.

  2. Download the comprehensive.xuf file from the email you received from CaseWare.

  3. Open Windows Explorer and copy the file to the following folder path:

    %userprofile%\Documents\CCH\T2 Taxprep 2017-1\Xpress

You have installed the required Xpress filter, and will be able to locate it from the TaxPrep interface.

Exporting from TaxPrep

With the Xpress filter installed, you can now export your data in the required .csv format.

Exporting TaxPrep data in .csv format

  1. Open the desired client file in TaxPrep.

  2. Select File | Export Tax Return….

  3. In the Export Tax Return dialog, select Xpress Filters......

  4. In the Cells to export pane, select comprehensive, then select OK.

  5. Make sure that the File Type is set to .csv, then select OK.

  6. Enter a file name and path, then select Save.

Sanitizing the Data

Before you send the data to CaseWare, please sanitize the data to remove all proprietary references. This will involve installing Node.js and running a sanitization script file.

Installing Node.js

  1. Go to the Node.js site and download the appropriate install package.

  2. Run the install package and complete the installation wizard.

  3. Download and unpack the SanitizeCSV.zip package anywhere on your computer.

The sanitization script

The sanitization file contained in the SanitizeCSV package runs a Node.js script that produces two sets of sanitized files:

  • The files saved to the /output folder have all proprietary information removed, but keep the original filenames intact.

  • The files saved to the /output_anonymized folder have all proprietary information removed, and anonymize the filenames. These are the files you will send to CaseWare for testing.

Running the sanitization script

  1. Copy your .csv output files to the input subfolder of the extracted SanitizeCSV folder.

  2. In the main SanitizeCSV folder, double-click run.bat. This will open a command prompt window with a confirmation message.

  3. Press Enter to run the script.

  4. Open a file in the /output_anonymized folder in a text editor such as Excel to ensure that all proprietary data has been removed and that the filenames have been anonymized. Repeat as desired for the remaining files.

  5. In Windows Explorer, compress the /output_anonymized folder into a .zip file.

    6. Note: Keep the files in the /output folder for reference and testing purposes.

  6. Send the compressed data to your point of contact at CaseWare.
