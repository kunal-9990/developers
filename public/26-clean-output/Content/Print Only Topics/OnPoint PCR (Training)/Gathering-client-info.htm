Gathering client info

You have completed all of the steps in the engagement acceptance stage:

  • completed the client acceptance and quality acceptance checklist documents

  • reviewed and finalized the draft version of the engagement letter

  • sent a copy of the engagement letter to the client to be signed

  • added the signed engagement letter to the engagement file

Now you will want to import the client’s trial balance to proceed with your review.

Submit the PBC Document Requests query

Open the Documents page, and in the Fieldwork section, open 2-930 PBC Document Requests - Engagements Subsequent to Initial Year. This query document includes a list of file requests for files and information that could be relevant for a review engagement.

Since there may be documents not relevant for this client or engagement, you can edit the query to make it more manageable for both you and the client. At the top of the document, select the Edit icon ().

All of the sections in this query document are now available for editing. You can select a text area to update the request description, you can use the Reorder icon () to move a procedure to a different place in the document, or you can use the Delete icon () to remove individual procedures or entire sections.

Note: When you delete a procedure or a section in a query, a confirmation dialog is displayed warning you that deleted procedures cannot be reinstated. If you remove content from a query by mistake, you can add a new copy of the original query to your engagement file on the Documents page.

Make the following changes to the document:

  1. In the text area of the first procedure, enter Chart of accounts and a delivery date under the Summary of requested items. Remove the other numbered lines from the summary.

  2. In the Current Year Transactions section, in the text area of the Chart of accounts procedure, replace the insert date placeholder text with the client’s year-end date.

When you are done, close editing mode by selecting the Edit icon () again. Change the status to Open to display the Open Query dialog. Choose a due date, and enter the following custom email message:

Please attach the chart of accounts for this year's review.

Select OK to open the query.

Once the client attaches their file, you can review the trial balance before accepting the procedure. In the query, you can select the file name to open it.

Note: When you attempt to open an office file in OnPoint, you will be prompted to confirm that you want to open the file in Office desktop.

Once you have confirmed that client’s file contains the trial balance information that you need, you can accept the procedures and complete the query. Select the Responded icon beside the Chart of accounts procedure and choose Accept.

Once you have accepted the procedure, change the query status from Responded to Complete.

When you go back to the Documents page, you will see that the trial balance file has been added to the Permanent File section under Current Year Transactions.

Import the trial balance

Now that you have added the client’s trial balance file to the engagement file, you can import the trial balance information. From the top menu, open the Trial Balance page.

Select the Import button, then choose Excel or CSV Import. A new dialog is displayed prompting you to select a file by either dragging and dropping it from your desktop, by browsing for it, or by selecting a file in the engagement.

When you choose Select file from engagement, you will be able to select the Excel file that you added to the engagement file from the client query. Select the file from this dialog, and the Column Mapping dialog is displayed next.

For each column in the import file, select a field to import the values in that column to. For the sample file, import the columns as follows:

Procedure Response
Account Number Account Number
Description Account Name
Current Year Current Year
Prior Year Prior Year
Prior Year Two Prior Year 2

At the bottom of the dialog, select the Import button.

The contents of the trial balance file are now imported into the engagement file. The net income is displayed at the top corner of the window, and if the trial balance is out of balance, warnings are displayed at the top of the relevant columns.

Modify the trial balance

When you import a trial balance, OnPoint will map as many of the accounts as it can, but in some cases, you may need to evaluate accounts and assign them manually. At the top of the page, in the trial balance menu, notice that the Assign accounts button now has a number in parentheses.

This indicates that one account is not currently assigned to a group. Select the button to open the Assign accounts page.

On this page, one account, Cash, is displayed in the Unassigned column.

Select this account to move it into the Selected Accounts section. When you do, OnPoint displays a few group suggestions based on the account name, “Cash”. To assign it to the Cash group, select the green Assign icon that is displayed beside the suggestion.

You can also drag and drop accounts on this page to assign them to a new group. In the Groups column, the account Bank - Savings is currently grouped under Cash and Cash Equivalents, but Bank - Checking is grouped one level lower, under Cash. Select Bank - Savings and drag it one line down, to the Cash group. When the group line is displayed with a green background, you can drop the account to assign it to the new group.

Return to the trial balance page. You can see that your changes are reflected here, as there are now three accounts in the Cash group.

Annotate the trial balance

While you are reviewing the trial balance, you can annotate accounts. From the trial balance menu, select the View button.

You can select different options here to change the information displayed on the trial balance. First, select Variance. You will see a new column that displays the variance between the final balance in the current period and the final balance from the prior period.

Notice that for your sample trial balance, there is no Prior value for the Bank - Savings account. You ask the client about this, and they inform you that the cash in the savings account came from sales that happened very late at the end of the year. You can add an annotation to include this information directly on the trial balance.

Select the Add annotation button () on the Bank - Savings line. A new dialog is displayed.

When you select this option, you can choose a purpose for your annotation, link it to a document or to an adjustment, and add a note to describe the reason for the annotation. You may also see a separate area where you can choose from several annotations suggested by OnPoint PCR.

For this annotation, select Variance for the purpose, and for the note section enter “Late sales at end of year.” Save your annotation.

Now the trial balance displays an icon for the annotation type, and it includes the description from the note.
