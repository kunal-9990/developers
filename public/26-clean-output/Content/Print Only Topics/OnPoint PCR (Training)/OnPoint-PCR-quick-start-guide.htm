OnPoint PCR Quick Start Guide

This document serves as a quick reference for the common steps that engagement team members will perform when working in OnPoint PCR.

Set up CaseWare Cloud

When your firm first begins using CaseWare Cloud, you will need to perform some initial setup steps:

  • create entities for your clients
  • create accounts for the staff members in your firm
  • create accounts for your client contacts

Help:

  • Create entities
  • Add staff and contacts

Start an engagement

To start your OnPoint PCR engagement, you will need to create the engagement file and set up the file to prepare staff members to work on it:

  • assign staff members to the engagement
  • set the engagement properties

Help:

  • Set up an engagement file

Perform engagement acceptance

In a new OnPoint PCR engagement, there are preliminary checklist documents that you must complete at the start of the engagement. Based on the responses for these preliminary checklists, other checklists are added to the engagement and an engagement letter template is also created.

Once the engagement letter is created, you will edit the text in the letter, then submit a PBC (Prepared by Client) query to the client to ask them to attach and return a signed copy of the engagement letter.

Note: Document and query names listed below will vary slightly based on the engagement type (Preparation, Compilation, or Review)

  • complete checklist - 1-110 Engagement Acceptance & Continuance
  • complete checklist - Engagement Quality Acceptance & Conclusion
  • complete engagement letter
  • submit a PBC (Prepared by Client) query - Engagement Letter (query to client)
  • (For Clients) respond to a PBC query - Engagement Letter (query to client)

Help:

  • Complete a checklist
  • Add a letter or memo (see the Modifying text and formatting section)
  • Gather data for an engagement (skip to To send a client request to a client)
  • Complete a client request

Gather client info

To begin working on the engagement, you will need to add documents or information from the client. OnPoint PCR includes PBC queries that you can submit for clients to attach any relevant files or documentation for the engagement.

To prepare the trial balance for your engagement, you will:

You will need to submit a PBC query for the client's trial balance. Once the client has responded and provided you with the trial balance, you will need to import the trial balance into the engagement. You may also need to perform work on the trial balance, assigning groupings, posting adjustments, and creating review notes.

  • submit a PBC query (to request the trial balance)
    • if this is your firm's initial engagement with client, use 2-920 PBC Document Requests - Initial Engagements
    • if this is not your firm's initial engagement with client, use 2-930 PBC Document Requests - Engagements Subsequent to Initial Year
  • (For Clients) respond to a PBC query (client should attach trial balance file)
  • import the trial balance
  • assign groupings
  • post adjustments
  • add and resolve review notes

Help:

  • Gather data for an engagement
  • Add an account or group to the trial balance
  • Post adjustments to the trial balance
  • Add review notes (see the Add review notes to an engagement file (SE) section)
  • Complete review notes (see the Complete review notes in an engagement file (SE) section)

Perform fieldwork

With the trial balance imported into the engagement, you will be able to start the fieldwork stage. As part of fieldwork, you will:

  • work on checklists
  • submit PBC queries
    • to get client input on certain checklist procedures in a review engagement, use 2-910 PBC Requests - Review Inquiries.
  • sign off on documents
  • add annotations
  • add and resolve review notes
  • attach external documents

Help:

  • Work in an engagement file
  • Attach documents to an engagement file

Complete the engagement

At the conclusion of an engagement, you will create a financial statements document for the client where you explain your findings and present a conclusion. OnPoint PCR has a built-in template that makes it easier for you to prepare the financial statements. While preparing the financial statements, you will:

  • replace placeholder text
  • add and edit notes
  • add cross-references to notes
  • add headers and footers
  • print the financial statements
