When your engagement requires additional information from the client, you can request this information directly from Cloud. If you've set up a portal account for client contacts, they can then log in and respond to your requests. You can create requests for various things:

  • Request for information - submit questions to the client.

  • Document request - ask the client to upload one or more files.

  • Send deliverables - send files to the client.

There are two (2) methods for creating a request: as a file request from the engagement workspace in Cloud, or as a client request from a supported engagement file. Both request types can be assigned to a client contact, where they'll then receive a notification by email to respond.

Creating a client request from an engagement file (SE)

This section describes functionality only available with products on the CaseWareCloud SE platform.

Learn more about Smart Engagement products.

When your engagement tasks require information from your clients, you can create client requests in a supported engagement file. Client requests can be accessed by any clients who have a contact profile in Cloud and who have visibility to their engagement.

Create a client request

Client requests are a type of document for Smart Engagement. When you want to request information from your client, you must first add a new client request to the engagement file.

To add a client request to the engagement file:

  1. From the top menu, select Documents.

  2. Select the Add Document icon (), then select Query.

  3. Select a query type, and enter a name.

    4. Note: Note: Available query types will differ depending on your Smart Engagement product.

  4. Select Create Query.

You have created a new client request document. The client request is displayed, and you can begin reviewing and modifying the client request as necessary. If you created your client request from a template, then you may only need to make minor changes.

Note: Note: If you have created a blank client request, or if you are making major changes to your document, see Customize a client request for more information.

Send a client request

Once you have reviewed the client request and made any necessary changes, you can send the document to your client.

To send a client request to a client:

  1. At the top of the document, select the status drop-down menu, and change the status to Open.

  2. In the Open Query dialog, add more details about the client request. You can set a due date, choose assignees from your client contacts, and add a message to the notification email.

  3. Select OK to submit the client request.

Once the query is open, your client contacts can log in to Cloud and see the client request in the My Work section of their overview page.

Creating a file request from a client workspace

When your engagement tasks require information from your clients, you can create and track file requests and assign them to your point of contact at the client organization. You can either request a new file or an updated version of an existing file.

Create a new file request

To request a new file:

  1. Ensure that you have the Viewer role and your contact has the Entity Collaborator role, or equivalent privileges.

  2. From the Cloud menu, select Activities.

  3. Select New | File Request.

  4. Enter the appropriate Entity. A list of users with entity access will appear.

  5. Select the contacts you want to make the request of to add them to the recipient list.

  6. If applicable, set a Due Date, then select Next.

  7. Enter a subject heading and message detailing your request, then select Next.

  8. Optionally, you can enter an additional message which will appear in the body of the request email. Otherwise, select Complete.

You have created and assigned a new file request.

Request an update to an existing request

To request an update to a file:

  1. Ensure that you have the Viewer role and your contact has the Entity Collaborator role, or equivalent privileges.

  2. From the Cloud menu, select Activities.

  3. Select New | File Request.

  4. Enter the appropriate Entity. A list of users with entity access will appear.

  5. Select the contacts you want to make the request of to add them to the recipient list.

  6. If applicable, set a Due Date, then select Next.

  7. Select the attach file () icon, then select Existing to upload an existing file from Cloud or select Upload to upload a new file.

    Enter a subject heading and message detailing your request, then select Next.

  8. Optionally, you can enter an additional message which will appear in the body of the request email. Otherwise, select Complete.

You have created and assigned a request for an updated file.
