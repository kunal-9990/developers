CaseWare Cloud enables you to upload client data, produced in other accounting software, to your client workspace to run analytical tests on the data. These reports can then be saved to Cloud, downloaded to your computer or exported directly to Working Papers to be included in your engagement.

Importing from desktop accounting software

You can import to Analytics from a wide variety of desktop accounting packages.

You can import client data for analysis from a number of desktop accounting suites.

Downloading CaseWare Transformer

First, you'll first need to download the CaseWare Transformer tool. Transformer will enable you to convert your QuickBooks client file to a specially formatted .zip package that can be uploaded to Cloud.

The following is a list of the accounting software that you can upload to Cloud through use of the Transformer utility:

  • AuditXML 3.x
  • MyOB AccountRight Live 2013 - 2016
  • Pastel Partner 12.1-14.0
  • QuickBooks CA 2012 - 2017
  • QuickBooks US 2012 - 2017
  • Sage 50 Accounting (Peachtree - WIN) 2012 - 2017

Note: Note: If you have client data from another accounting software, you can still upload this data to Cloud for analysis. Simply export the data to a .CSV file (such as Excel), format the data according to specifications and upload it to Cloud. For more information, see Import client's financial data from a CSV file for analysis.

You can also download Transformer from the Analytics interface.

To download the Transformer tool:

  1. Ensure that you have the Editor role on the file or equivalent privileges.

  2. From the Cloud menu, select Analytics and then select the appropriate Analytics engagement.

  3. Select Add Data File | Download Transformer Tool.

After you've downloaded the .exe file, open it and complete the steps in the wizard to install CaseWare Transformer.

Converting data to .zip format

The procedure for converting data to .zip format differs depending on your accounting software package.

Converting QuickBooks data to .zip format

With Transformer downloaded and installed on your computer, you can now convert your QuickBooks files for use in Analytics.

To use the Transformer tool to convert data from QuickBooks CA or QuickBooks US:

  1. Use QuickBooks to open the client file you want to convert.

  2. Open CaseWare Transformer.

  3. In the Software Package field, select QuickBooks.

  4. Next to the Export Path field, click Browse and select the location you want to save the converted file to.

  5. Click Export.

  6. Specify the fiscal year and click OK.

The QuickBooks file you've selected is converted into a .zip format recognized by Analytics.

Converting AuditXML data to .zip format

With Transformer downloaded and installed on your computer, you can now convert your AuditXML files for use in Analytics.

To use the Transformer tool to convert data from AuditXML:

  1. Open CaseWare Transformer.

  2. In the Software Package field, select AuditXML.

  3. Next to the Data Path field, click Browse and select the client file folder you want to import.

  4. Next to the Export Path field, click Browse and select the location you want to save the converted file to.

  5. Click Export.

The AuditXML file you've selected is converted into a .zip format recognized by Analytics.

Converting MyOB data to .zip format

With Transformer downloaded and installed on your computer, you can now convert your MyOB files for use in Analytics.

To use the Transformer tool to convert data from MyOB AccountRight Live:

  1. Open CaseWare Transformer.

  2. In the Software Package field, select MyOB AccountRight Live.

  3. Next to the Export Path field, click Browse and select the location you want to save the converted file to.

  4. Click Export. The Linking With MyOB Live dialog will appear.

  5. Click Select Client and specify the client associated with the file you want to convert.

  6. From the Select a company drop-down list, choose the client file you want to convert.

  7. Complete the fields and click OK.

The MyOB file you've selected is converted into a .zip format recognized by Analytics.

Converting Pastel Partner data to .zip format

With Transformer downloaded and installed on your computer, you can now convert your Pastel Partner files for use in Analytics.

To use the Transformer tool to convert data from Pastel Partner:

  1. Open CaseWare Transformer.

  2. In the Software Package field, select Pastel Partner.

  3. Next to the Data Path field, click Browse and select the client file folder you want to import.

  4. Next to the Export Path field, click Browse and select the location you want to save the converted file to.

  5. Click Export.

  6. Select whether or not the file will be imported to a rolled-forward file and click OK.

The Pastel Partner file you've selected is converted into a .zip format recognized by Analytics.

Uploading your converted client files to Analytics

With your QuickBooks file converted, you can now upload the client data to your Analytics engagement in order to produce Analytics reports on the data.

To add a Transformer zip file to Analytics:

  1. Ensure you have the Editor role or equivalent privileges.

  2. From the Cloud menu, select Analytics.

  3. Select the appropriate Analytics engagement.

  4. From the Data Files pane on the left, select Add Data Files. The Upload New File dialog opens.

  5. Select Select New File and browse your computer for the file you want to upload. Select OK to begin the upload process.

The client data contained in the .zip file is added to your Analytics engagement. An overview of the data is displayed in the engagement's Profile page.

Importing from online accounting software

We have begun to integrate Analytics with online accounting platforms, enabling you to upload data directly from these services without having to leave your browser or download the engagement files to your hard drive. These operations do not require the use of CaseWare Transformer. Currently, Analytics supports Xero Online; more platforms will be supported in future releases of Analytics.

Importing from Xero Online

If you use Xero Online to manage your client data, you can import these files directly into Analytics by logging into your Xero account and selecting the appropriate file. Xero imports currently only support up to one year of transaction data.

To import client data from Xero Online into Analytics:

  1. Ensure that you have the Editor role or equivalent privileges. To connect to the Xero file, ensure that you have a valid Xero account and that you've been assigned the Standard user or Auditor role on the file. For more on users and roles in Xero Online, see their help materials.

  2. In your Analytics engagement, select Add Data File | Import File From Xero.

  3. A new browser window will open with the Xero login page. Enter your account credentials and select Login.

  4. Select the appropriate client from the drop-down list, then select Allow access for 30 mins.

  5. Select the checkboxes next to the client files you want to import, specify a Current fiscal year end date, then select Import.

    Note: Note: The Current fiscal year end date affects how the data is displayed, as well as the defaults for date-based Analytics tests. All data in the file, regardless of date, will be imported.

The client file from your Xero account is now available for use in Analytics, and is displayed in the Data Files pane.

Importing from QuickBooks Online

If you use QuickBooks Online to manage your client data, you can import these files directly into Analytics by logging into your QuickBooks account and selecting the appropriate file.
