Adding tickmarks and annotations

This feature is only available with products on the CaseWareCloud SE platform.

Learn more about Smart Engagement products.

You can add annotations and tickmarks to each document in your engagement file. You can provide a status to a particular entry, add commentary, or link to other relevant documents.

Each annotation has a unique icon:

  • Agrees to... ()

  • Comes from... ()

  • Goes to... ()

  • Calculation Verified ()

  • Addition Checked ()

  • Link ()

  • Note ()

  • Variance ()

To add a tickmark or annotation to a document:

  1. From the top menu, select Documents.

  2. Select a document to open it.

    Note: You can add annotations to checklists and to PDF documents.

  3. In your document, select the Add annotation icon () for the document section you want to annotate.

  4. Enter all of the information for your annotation.

    Note: You can select a suggestion to automatically apply the suggested annotation.

  5. Select Save.

The annotation is added to the entry. The unique icon for each annotation type displays next to the associated section of the document. Selecting the icon displays pop-up text with a summary of the annotation content.
