Managing sync copies

  • Opening sync copies
  • Deleting sync copies
  • Clearing synchronization information from Cloud

This feature is only available with CaseWare Working Papers.

Learn more about Working Papers.

Opening sync copies

Users will need to create a synchronized copy of the parent file on their local drive to work on the file. To facilitate this operation, CaseWare Cloud provides a way to create or open an existing sync copy directly in the Cloud.

To open sync copies:

  1. Ensure you have the Editor or equivalent role.

  2. From the Cloud menu, select Working Papers.

  3. Navigate to the Working Papers SmartSync file you wish to open. Select it to open the File Details pane.

  4. Select the ellipsis button () and choose Open Sync Copy from the drop-down menu.

  5. CaseWare Working Papers will launch, download the file and create a synchronized copy. If you already have a synchronized copy, then the following prompt will appear and it will open the existing synchronized copy.

 

Deleting sync copies

To delete a Working Papers SmartSync file in CaseWare Cloud you must first delete any sync copies in Working Papers and then delete the file from Cloud.

To delete sync copies from Cloud:

  1. Ensure you have the Editor or equivalent role.

  2. From the Cloud menu, select Working Papers.

  3. Navigate to the SmartSync file you wish to abandon. Select it to open the file details pane.

  4. Select the ellipsis icon () and choose Abandon Sync Copy from the drop-down menu.

  5. Select the Sync copy you wish to abandon and select OK.

Clearing synchronization information from Cloud

The option Clear SmartSync Information is used to clear the _Sync folder of a Working Papers client file in Cloud.

To clear synchronization information from Cloud:

  1. Ensure that all sync copies are abandoned before clearing synchronization information from the Cloud.

  2. Ensure you have the Editor role or equivalent privileges.

  3. From the Cloud menu, select Working Papers.

  4. Select an entity from the Entity Selector.

  5. Open the details of the Working Papers file whose _Sync folder you want to clear.

  6. On the Details pane, click SmartSync Summary and make sure that all existing sync copies of the file have been removed.

  7. Select ellipsis ( ) | Clear SmartSync Information.

  8. When asked to confirm the action, click Yes.
