CaseWare Time can be integrated with Cloud to provide additional features and functionality through our online portal. With Time connected to Cloud, you can track your time, manage staff, and create reports directly from your internet browser.

Integrating Time (Desktop) with Cloud

The process of Integrating Time with Cloud is called merging. Merging will take the important data from your Time file and move it to Cloud. Once merged, this data is updated continuously over the internet.

To integrate Time (Desktop) with Cloud:

  1. Ensure you have Time 2016 or higher.

  2. Launch the Time desktop application and log in with the Administrator account. The default sup account cannot be used.

  3. Open the file you wish to merge with Cloud.

  4. Protection must be enabled for the file prior to merging. To enable protection, select Tools | Protection | Turn Protection On.

  5. To begin the merging process, while still in the Tools menu, select Time on CaseWare Cloud Setup | Upload data to CaseWare Cloud from Time.

  6. View the wizard and ensure all prerequisites are met. Select Next to continue.

  7. Enter your Firm URL and login information in the applicable fields. A Cloud account with Staff Admin role or equivalent privileges is required to authorize the merge. Once entered, select Next.

  8. The merged data is displayed. Click Finish to complete the wizard.

Your Time file is now integrated with Cloud. Accessing the file in the Time application will now display an Online status and updates will automatically be pushed to Cloud. Note that only one Time file may be merged with Cloud at a time.

Manually synchronizing your Time and Cloud data

When your Time file is linked to CaseWare Cloud, data (including your Company Profile, Work Codes, Staff Rates and Contacts) is automatically synced when changes are made. If you have set the file status to Offline, or need your changes updated immediately, you can perform a Manual Merge. Manual merging will force the program to combine the current data between Time and Cloud.

To manually synchronize Time and Cloud data:

  1. Open the file in the Time application that is connected to Cloud.

  2. Select Tools | Merge Time and CaseWare Cloud.

The current Time and Cloud data is updated for immediate use.
