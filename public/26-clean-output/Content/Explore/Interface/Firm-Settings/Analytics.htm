Manage CSV File Types

This page lists the possible types that can be assigned to CSV files when they are being imported into Analytics. You also have the option to add a Custom Type, which will appear in the Type drop-down in the File Import dialog in Analytics.

For more information on importing date from a CSV file into Analytics, see Import client's financial data from a CSV file for analysis.
