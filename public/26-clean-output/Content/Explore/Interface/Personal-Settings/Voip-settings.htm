VoIP Settings

From My Settings | Other | VoIP Settings, you can enable VoIP integration in Cloud. When enabled, all telephone numbers are converted into links that can be dialed by your computer. Clicking on these links will launch your default VoIP application. A VoIP application must be installed on your computer for this feature to work.

Supported phone clients include:

  • Skype
  • Google Voice
  • Microsoft NetMeeting

If no country calling code is specified in a phone number, the Default country calling code is prefixed to the phone number. Ask your Cloud administrator if this setting is specified in Settings.

Because your browser must make a request to an application installed on your computer, a warning appears to obtain your permission to access this application. Allow this operation to proceed.
