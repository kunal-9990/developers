Known issues in CaseWare Cloud

This topic describes known issues when using CaseWare Cloud.

Error when importing entities from a CSV file

Importing 95 entities or more from a CSV file will result in an error. To work around this issue, import entities in batches of 94 or less.

CSERV-1132

Errors with removed Time users

Currently, users who are logged in and have their Time access revoked by an administrator will get multiple error messages when they attempt to access Time.

CORE-46707

Internet Explorer entity selection issue

Users accessing Cloud using Internet Explorer may experience an issue in the Engagements page when attempting to create a new engagement. When users select New and the engagement type, they are asked to select an entity from a drop-down list, but it takes several clicks before the entity is selected and the drop-down list clears. To work around this issue, use an alternative browser (such as Chrome or Firefox) when using the Engagements app.

COLB-444

Error messages on the Working Papers page

Inaccurate error messages are sometimes displayed when abandoning a sync copy or when clearing sync information on the Working Papers page. To work around this issue, wait a few minutes and then refresh the page to confirm the intended action took place.
