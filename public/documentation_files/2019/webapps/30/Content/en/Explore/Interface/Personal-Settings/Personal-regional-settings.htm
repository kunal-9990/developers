Personal regional settings

From My Settings | Other | Regional Settings, you can override your organization's Regional Settings for Date and Time formats and for Numeric values.

Name Description
Date and Time

Sets the appearance of the date and time format:

  • Preview - a preview of the date and time format using the selected options.
  • Use Organization Default - clear to override the date format set by the organization.
  • Date Format - the date format.
  • Use Organization Default - clear to override the time format set by the organization.
  • Time Format - the time format.
Numeric

Sets the appearance of numeric values:

  • Preview - a preview of a positive number using the selected options.
  • Use Defaults - clear to override the decimal symbol and digit grouping symbol set by the organization.
  • Decimal Symbol - the symbol used to denote decimal values.
  • Digit Grouping Symbol - the symbol used to group digits (e.g. 10,000).
