CaseWare Cloud contains multiple ways to organize your files to increase productivity.

Moving files in bulk

Let’s say you uploaded all your files in an internal entity and would like to move all finalized drafts to the client’s entity. In CaseWare Cloud, you can move files in bulk to an entity or a folder.

To move files to another entity:

  1. Select All Entities or the entity with the files you want to move from the Entity Selector.

  2. Select the file (or files) you want to move to another entity.

  3. In the file details pane, select More Actions () | Move to Entity.

  4. Choose the folder you want to move the file to and select OK.

  5. A prompt to confirm the move will appear. Select Yes.

 

 

You have moved files in bulk to another entity.

Setting file retention

You can set retention on files to remove any unwanted items from the system within a certain number of days, months, or years. As a result, this will help ensure that the next year file contains only the finalized items and reports.

To set file retention period:

  1. Ensure you have the Editor role or equivalent rights.

  2. Select the file you want to set the retention for.

  3. Select the More Actions () | Set Retention.

  4. Select the retention start date, how long to retain the file, and any comments if necessary.

  5. Select OK to confirm. The file details pane will automatically be updated with the retention status information.

You have set file retention.

Removing unwanted files in bulk

You can remove unwanted files in bulk that are not required in the finalized file.

To delete several files at once:

  1. Select the files you want to remove.

  2. In the file details pane, select the More Actions () | Delete.

  3. Confirm deletion of the files. If you removed a file in error, you can recover it from the Recycle Bin. See Recover a previous version or deleted file for more information.

 

You have removed unwanted files in bulk.
