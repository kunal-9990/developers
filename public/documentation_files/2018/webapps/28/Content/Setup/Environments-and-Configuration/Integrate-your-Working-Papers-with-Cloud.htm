Integrate your Working Papers with Cloud

This feature is only available with CaseWare Working Papers.

Learn more about Working Papers.

Integration with CaseWare Cloud provides additional functionality to Working Papers. Connecting these products enables the use of Cloud as a file management and storage center, while also providing new ways to track and complete your engagement online.

To integrate Working Papers with Cloud:

  1. Ensure you have Working Papers 2016 or higher.

  2. Launch the Working Papers application.

  3. On the File menu, select Options | CaseWare Cloud.

  4. Ensure Integrate with CaseWare Cloud is selected. In the Server field, enter the URL of your organizationâ€™s CaseWare Cloud site. You do not need to enter the web protocol (http:// or https://). Click OK.

  5. A CaseWare Cloud Login prompt is displayed. Enter your User name and Password, then select OK.

Your copy of Working Papers is now integrated with Cloud.
