Import Client Files

From Settings | Working Papers | Import Client Files, you can batch import multiple client files at once.

Number Name Description
Entity selection method Select the options for how you would like Cloud to assign Cloud entities to the files.
File Name The name of the file.
Entity The selected Cloud entity associated with this engagement.
Client Name The information in the Client field of the Engagement properties.
Operating Name The information in the Operating Name field of the Engagement properties.
Client Number The information in the Client # field of the Engagement properties.
Home Page The information in the Home Page field of the Engagement properties.
File Size The size of the file.
Path The folder path of the client file.

Working Papers Security

From this section, you can create custom security roles for synced Working Papers files, or import an existing security configuration. For more information, see Controlling access to Working Papers files.
