Supported third-party desktop accounting software

This feature is only available with products on the CaseWareCloud SE platform.

Learn more about Smart Engagement products.

You can import client data from most third-party desktop accounting software for use with a variety of CaseWare Cloud products. To learn more about importing client data, see Importing from third-party desktop accounting software.
