Engagements page

Use the Engagements page to review engagement progress.

From this page, you can see a list of all the engagements for your organization that you have permission to view. This includes Working Papers files, CaseWare Cloud engagements and other supported engagement types.

Access the Engagements page from the Cloud menu.

Select My Work to see the engagements assigned to you.

If you are managing engagements for your organization, you can filter the list of engagements by type, and see the status for all of your engagements at a glance. You can also see any missed milestones by selecting Overdue.

You can select any engagement from the list to view more details. Depending on the engagement type, you can review key dates, outstanding issues and the number of staff hours and total amount billed for the engagement. All of this information is displayed in one place, making it easy to monitor the progress of each of your engagements.

The Document Status bar has these areas:

  • Outstanding - The number of documents that have not been prepared or reviewed.

  • Prepared - The number of documents that have been marked as Prepared () in the engagement from the open document's page.

  • Reviewed - The number of documents that have been marked as Reviewed () in the engagement from the open document's page.

Note: The Document Status is updated every five minutes.
