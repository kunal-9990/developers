Add the CloudBridge document to Working Papers

Before you can export data from a Working Papers engagement, you will need to add the CloudBridge (CLBG) document to the Document Manager.

To add the CLBG document to Working Papers:

  1. Ensure your copy of Working Papers is integrated with your Cloud firm. To learn more, see Integrate Working Papers with Cloud.

  2. Ensure that you have installed CloudBridge. To download CloudBridge, see Import engagement data from CaseWare Working Papers to Fusion RCT.

  3. Open your client file in Working Papers.

  4. From the Document tab, click From Library | CloudBridge. The Document Library dialog displays.

  5. Click CLBG and click Add.

The CLBG document displays in the Document Manager. You can now use the CLBG document to export your engagement data.

Export engagement data from Working Papers

After you add CLBG document to a Working Papers engagement, you can export your trial balance and general ledger tax data to Fusion RCT.

To export your engagement data to Fusion RCT using CloudBridge:

  1. In the Document Manager, double-click CLBG. The CLBG document displays.

  2. In the Step 1 section, choose an appropriate entity and engagement.

  3. In the Step 2 section, choose the appropriate trial balance data to export.

  4. (Optional) In the Step 2 section, click the Use remap file check box to remap your current engagement automatically. Click Choose file to select an appropriate remap file.

    Note: If your engagement uses the default Working Papers mapping or was created with the PEG template, use the remap file located in the folder where you installed CloudBridge.

  5. (Optional) In the Step 3 section, choose the appropriate trial balance accounts and tax worksheets to export GL data. The Step 3 section will only display if your engagement file contains tax data in the general ledger.

  6. Select Transfer Data.

Your exported engagement data is available in Fusion RCT. If you exported general ledger data, you can see all of your updated tax sheets by selecting Documents | More Actions ( ) | Hidden Items in your Fusion RCT engagement. Some of your tax sheets may be visible by default.
