Release history - Time

The following describes the features and improvements made in previous releases of CaseWareCloud Time (select Help | About to find Cloud's version number).

What's new - Time 27.0

The following describes the features and improvements made for CaseWareCloud Time 27.0 (select Help | About to find Cloud's version number).

Automatically itemize invoices

Staff can now automatically itemize an invoice based on time and expense entries. Automatic itemization significantly speeds up the process of creating a detailed invoice in Cloud. To learn more, see Itemize an invoice automatically.

Create fixed fee invoices

The Fixed Fee invoice type is now available. Staff can use a Fixed Fee invoice to bill clients without needing to create time or expense entries. To learn more, see Draft an invoice.

Hide invoices on the Billing page

Staff can now hide invoices that are no longer required by setting the status to Closed. This helps ensure that only active invoices are displayed. To learn more, see Close an invoice.

General improvements

Several improvements have been made for the latest release.

Billing

  • Fixed an issue where search could fail when many client entities have similar names.

  • Fixed an issue where a user with insufficient permissions could see unusable invoice options.

  • Fixed an issue where an invoice template could be deleted after an invoice was created using that template.

Settings

Fixed an issue where email tags were displayed incorrectly in the email log.

Time

Fixed an issue where French- and Spanish-language users could not select any Monday using the Dates filter.

QuickBooks Online integration

Fixed an issue where exported invoices did not include complete address information.

What's New - Time 26.0

The following features and improvements were made for CaseWareCloud Time 26.0.

Ability to export invoices to QuickBooks Online

Administrators can now link Cloud with an existing QuickBooks Online (QBO) account. Invoices can then be exported directly to QBO to track payments and manage receivables. To learn more about exporting invoices, see Integrate Cloud with QuickBooks Online.

Ability to disable GL features

Administrators can now disable general ledger (GL) features. This ensures that only necessary information is collected during the billing process. If an organization has never used GL features, GL will be disabled during migration to 26.0. New organizations will have GL features disabled by default. Administrators can enable or disable GL features by selecting Firm Settings | Time | General | Time and Expense Settings | Enable general ledger.

Billing page improvements

A number of improvements have been made to the Billing page, including:

  • an option to send invoices directly from the Invoice Wizard,

  • search and filter functionality on the WIP tab and

  • the ability to revert the status of invoices from Sent to Client to Draft.

General improvements

Several improvements have been made for the latest release.

Analysis

  • Fixed an issue where certain Analysis reports did not display values in the Type column correctly.

  • Fixed an issue where some sorting options were not available for certain Analysis reports.

  • Fixed an issue where some Time Summary reports would export with incorrect formatting.

Billing

  • Fixed an issue where users in certain time zones could not set an invoice billing date to the first day of the active period.

  • Fixed an issue where invoices were deselected after being sorted on the Invoices tab.

Firm Settings

  • Fixed an issue where inactive work codes could be marked as active.

  • Fixed an issue where enabling SharePoint could affect Time app permissions.

What's new - Time 24.0

The following features and improvements were made for CaseWareCloud Time 24.0.

Time license management

You can now grant access to the Time app and its tools on the Apps tab in the Create and Edit staff dialogs. This makes it easier to manage Time app access, bringing it in line with other Cloud apps that require licenses. The permissions for Time and Expense that were under Security | Role Permissions on the Firm Settings page have been removed.

You can now see when a staff member's Time license status is Unpaid in the App section of their profile, making it easier for you to manage your Time licenses.

Transfer time and expense entries

You can now transfer time and expense entries between entities. This enables you to consolidate time and expense entries when you need to delete an entity, or move entries that were added to the wrong entity by accident. Entries can be moved to another entity, or a specific engagement associated with that entity. To learn more, see Transfer time or expense entries.

Improved invoice customization

You can now add multiple line items to invoices. Invoice line items can be created using pre-existing work codes and billing codes, or by entering a custom line item. To learn more about sending an invoice to a client, see Send an invoice to a client.

Office 365 calender integration

You can now integrate Time with Office 365 to generate pending entries based on events in your Outlook calendar. To learn more, see Integrate Office 365.
