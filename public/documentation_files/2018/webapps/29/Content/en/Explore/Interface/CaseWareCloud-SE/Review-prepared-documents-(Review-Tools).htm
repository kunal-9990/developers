Review prepared documents (Review Tools)

This feature is only available with products on the CaseWareCloud SE platform. Available SE products include OnPoint PCR and CaseWare ReviewCompTax.

Review tools apply visual markup to the document, making potential issues with content stand out on the page so you can complete your review more quickly.

Review tools are available in checklists, letters and financial statements.

Review content changes in a letter or a checklist

The following review tools are available in letters and checklists:

Tool Description

Added

Content that was added by staff members. Ensure that it is relevant and necessary for the engagement.

Modified

Changes made to the default product content by staff members. Ensure that the changes are correct and that content still complies with guidelines and standards.

Overridden visibility

Content that has been manually hidden or forced visible by staff members. Confirm that the hidden content does not apply to the engagement. Confirm that the forced visible content does apply to the engagement.

Optimized out

Content that has been automatically hidden based on its visibility conditions. Confirm that the hidden content does not apply to this engagement.

Non optimal answers

Responses that may indicate a problem on the engagement. Confirm that the chosen response is correct and determine whether annotation or supporting documentation is required.

Modified after sign-off

Responses that were changed after a staff member had signed off on the query.

Not signed off

Steps that have not been signed off by staff members.

To review changes in a letter or checklist:

  1. At the top right of the document, select the Review icon ().

    2. Review tools display in the left panel.

  2. You can display different review content by selecting the appropriate checkboxes in the left pane.

Review content changes in the financial statements document

The following review tools are available in the financial statements document:

  • New content () - Content that's been added by staff members.

  • Excluded content () - Content that's been excluded by staff members.

  • Text area changes () - Modifications made by staff members in text sections that are part of the default content that comes with the product.

  • Dynamic table changes () - Modifications made by staff members to dynamic tables that are part of the default content that comes with the product.

To review changes in a financial statements document:

  1. Select Toggle Review Tools () at the Document Map.

    The number of changes in each financial statements area displays (if available).

  2. Select an area to review the changes.

    Changes in the selected area are marked with the applicable review tool icon.

To filter the displayed changes:

  1. Select the Filter Review Tools ().

    2. A popup displays showing check boxes for each review tool.

    All check boxes are selected by default.

  2. Clear one or more check box to specify which changes you want to review in the document.
