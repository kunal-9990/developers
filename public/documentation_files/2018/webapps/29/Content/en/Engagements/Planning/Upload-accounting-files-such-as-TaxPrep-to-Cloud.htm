Upload accounting files such as TaxPrep to Cloud

If you manage your tax work in TaxPrep, you can upload your TaxPrep files to Cloud in order to track and assign work on these files as you would with any other file type.

The procedure for adding a TaxPrep file is the same as adding any other file. After the file is uploaded, Cloud will automatically create an Other Engagement and store the TaxPrep file there.

To upload a file:

  1. Ensure that you have the Editor role or the equivalent privileges.

  2. From the Cloud menu (), select Files.

  3. Select the appropriate entity using the Entity Selector.

  4. Select New, and then choose Upload. The File Upload dialog is displayed.

  5. Use the Select Files button, or drag and drop a file to upload it.

  6. In the Share Settings dialog, choose sharing options for the file. Select OK.
