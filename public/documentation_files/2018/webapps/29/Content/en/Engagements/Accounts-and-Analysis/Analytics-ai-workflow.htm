Analytics.AI workflow

This feature is only available with CaseWare Analytics.AI, part of the CaseWare Cloud suite.

Learn more about Cloud.

The following is an example of a typical Analytics.AI workflow. Your exact workflow will vary depending on the selections that are made in the Optimizer.

  1. From the Cloud menu (), select Analytics ().
  2. Create an engagement in the usual manner.
  3. From the Documents page:
    1. Complete the 0-1 Optimizer document.
    2. Select Upload Trial Balance to upload your trial balance.
    3. Select Upload Transactions. Analytics.AI starts running tests in the background.
  4. Optionally, examine the Data Completeness page.
  5. If you opted to use influencers in the Optimizer, complete the Influencers document.
  6. Optionally, examine the Analytical Review page.
  7. From the Documents page, click on the Cloud Analytics link to see Analytics.AI test results.
  8. Export results for audit evidence.
