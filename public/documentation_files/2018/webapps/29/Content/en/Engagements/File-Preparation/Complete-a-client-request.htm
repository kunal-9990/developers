Complete a client request

This feature is only available with products on the CaseWareCloud SE platform. Available SE products include OnPoint PCR and CaseWare ReviewCompTax.

Respond to a client request

Once you have shared your client request, your contact can see the request on the My Work pane when they log in to Cloud. They can select the client request to display it and enter their responses and comments.

Your contact can mark the entire client request as responded by selecting Mark as responded.

Contacts can also mark individual queries within a client request as responded by selecting the Responded icon ().

Once your contact has responded to the client request, the Respondedicon () is available next to the client request on your list of documents. When you open the client request, you can see the time of the response and the name of the contact who responded to each query.

Reopen a client request

If you need more information or corrections on the document, you can reopen the request.

To reopen a client request:

  1. From the top menu, select Documents.

  2. Select the client request from the document list.

  3. In the document, select Reopen.

Reopen a query in the client request

If there are problems with some of your client's responses, you can reopen individual queries.

To reopen a client request query:

Select the Prepared icon () next to the query, and then choose Decline.

Once the client request query has been reopened, the contact for your client will see the new status. They can access the client request from Cloud to address your comments or to add missing information.

Accept a client request

If the client has provided all of the requested information, you can change the status to Complete.

To accept the client response:

  1. From the top menu, select Documents.

  2. Select the client request from the document list.

  3. In the document, select Complete.

The request is now accepted. The information provided in the document is finalized and can no longer be changed. Completed client requests are no longer visible to contacts in Cloud.
