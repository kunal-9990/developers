Add annotations

This feature is only available with products on the CaseWareCloud SE platform. Available SE products include OnPoint PCR and CaseWare ReviewCompTax.

You can add annotations in the trial balance, checklists, adjustments or PDF documents. You can provide a status to a particular entry, add commentary, or link to other relevant documents.

Once you add an annotation, one of the following icons displays on the page to represent the purpose of the annotation:

  • Agrees to ()

  • Comes from ()

  • Goes to ()

  • Calculation Verified ()

  • Addition Checked ()

  • Link ()

  • Note ()

  • Variance ()

Annotate a checklist

You can add an annotation in procedures and subprocedures in a checklist document.

To add annotations in a checklist:

  1. Open the engagement file and select Documents

  2. Select a checklist document to open it.

  3. Go to the procedure or subprocedure that you want to annotate, then select the Add annotation icon ().

  4. Enter the information for the annotation.

    1. Select the purpose of the annotation.

      Note: You can select a suggestion to automatically apply the suggested annotation.

    2. Link the annotation to document or an adjustment (if applicable).

    3. Enter notes or comments.

  5. Select Save.

The icon corresponding to each annotation purpose displays in the document. If you select the icon, a pop-up displays showing a summary of the annotation.

Annotate the trial balance

You can add an annotation for accounts and groups lines in the trial balance.

To add annotations in the trial balance:

  1. Select the line that you want to annotate.

  2. Select Add Annotation ().

  3. Enter the details of your annotation.

    1. Select the purpose of the annotation.

    2. Link the annotation to document or an adjustment (if applicable).

    3. Enter notes or comments.

  4. Select Save.

    The icon corresponding to the annotation purpose displays next to the annotated line. If you select the icon, a pop-up displays showing a summary of the annotation.

Annotate an adjustment

You can add an annotation in for the entire adjustment entry or for each line in the entry.

To add annotations in an adjustment:

  1. Go to the Adjustments page.

  2. Select Edit () next to the adjustment entry that you want to annotate.

    The Adjustment Details dialog opens.

  3. To annotate the entire adjustment entry, select Add Annotation () at the top right corner of the dialog and enter the annotation details.

  4. To annotate a line in the adjustment entry, go to the Details section and select Add Annotation () next to the line.

  5. Enter the details of your annotation.

    1. Select the purpose of the annotation.

      Note: You can select a suggestion to automatically apply the suggested annotation purpose.

    2. Link the annotation to document or an adjustment (if applicable).

    3. Enter notes or comments.

  6. Select Save.

    The icon corresponding to the annotation purpose displays in the dialog. If you select the icon, a pop-up displays showing a summary of the annotation.

Annotate a PDF document

You can add annotations anywhere in a PDF document.

To add annotations to a PDF document:

  1. From the top menu, select Documents.

  2. Select the PDF file from the document list.

  3. In the viewing pane for the PDF file, select the area where you want to add annotation.

    The Add Annotation icon () displays.

  4. Select Add Annotation () and enter the information for your annotation.

  5. Select Save.

    The icon corresponding to each annotation purpose displays next to the annotated section of the document. If you select the icon, a pop-up displays showing a summary of the annotation.
