Link a query to a checklist procedure

You can link a query to a related procedure in a checklist to help users complete the checklist document.

For example, if there is a question in a checklist related to the client’s business that you know users won’t have an optimal answer to and it’s better they request that information from the client. So, you create a Query template and add this request, then link it to the related question in the checklist document. Once users collect the required information from the client, they add it in the checklist.

To link a query to a checklist:

  1. Go to the query that you want to link to a checklist.

  2. Select the query, then select More actions () | Link to Checklist.

  3. Select the search field to display a list of the available checklists in the template.

    4. You can enter a keyword in the search field to filter the list. You can also select Show hidden documents check box at the bottom to include hidden checklists in the list.

  4. Select a checklist document to link the query to.

    5. Once you add a checklist, another search field displays. It allows you to select specific procedures in this checklist to link the query to.

  5. Select the search field to display a list of the procedures in the checklist.

    6. You can enter a keyword in the search field to filter the list. You can also select Show hidden procedures check box to include hidden procedures in the list.

  6. Select a procedure to link the query to.

  7. Select Save.

To remove the link, select the query, then select More actions () | Link to Checklist. Once the dialog opens, select Delete.
