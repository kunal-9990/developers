Workflow (how to author financial statements?)

The following steps summarize the recommended workflow for authoring financial statements. You can use this workflow for guidance.

  1. Add document guidance

  2. Add areas

  3. Add content in areas

    • Add text

    • Add tables

    • Set up the statement of Cash Flows

    • Add a page break

  4. Set up the financial statements notes

  5. Add content guidance

  6. Add a header or footer

  7. Rearrange areas

  8. Set up the print options

  9. Add visibility settings
