Define engagement glossary terms

You can define the terms in the engagement glossary and the display logic for each term using conditions.

For example, if the user answers Yes to Does your accounting practice have only one practicing partner? procedure in the 010-Optimizer checklist, then the output term is My. If the answer is No, the output term is Our.

To add terms in the Engagement Glossary:

  1. Select Information () | Engagement Glossary.

    2. The Engagement Glossary displays. At this point, it’s empty.

  2. Select Add Group ().

  3. Enter a group name, for example Terminology.

    4. The Add term button is now available on the page.

  4. Select Add term to open the Term Settings dialog.

  5. Enter a Description, then select the Add Condition drop-down list.

    6. Different setup details display based on the condition you choose:

    Condition type

    Condition setup

    Response

    Document - select the source document for the response

    Procedure -  select the procedure for which the response is provided

    Condition - select the value of the response

    Output - select the text to be displayed when the condition is met

    Trial balance

    Financial Group - select the trial balance account or account group

    Condition - select how the name of the Income Statement displays based on the value of Net Income in the trial balance

    • Consistent → name displays as "Statement of Income"
    • Contrary → name displays as "Statement of Loss"
    • Inconsistent → name displays as "Statement of Income (Loss)"

    Output - select the text to be displayed when the condition is met

    Consolidation

    Condition - select the consolidation situation (Consolidated, Non-consolidated)

    Output - select the text to be displayed when the condition is met

    Reporting period

    Condition - select the value of the reporting period (Monthly, Quarterly, Semi-Annual, Annual)

    Output - select the text to be displayed when the condition is met

    Organization type

    Condition - select a country

    Output - select the text to be displayed when the condition is met

    Group of conditions

    Select this option if you want to add a group of conditions, where the text output is based on how the conditions in the group are collectively evaluated

    Group Condition - select one of the following values:

    When all conditions below are met -> all conditions must be true for the Group Condition to be met

    When any condition below is met -> any condition must be true for the Group Condition to be met

    Output - select the text to be displayed when the Group Condition is met

    You can add more than one condition type to trigger additional text outputs. They could be the same type or different types.

    To add another condition, select Add Condition. To add the same condition type select Copy Condition ().

  1. Once you’re done, select Save.

    2. If a condition (or conditions) is met, the corresponding output text you specified for that condition displays. If no condition is met, the #ValueEmpty# displays.
