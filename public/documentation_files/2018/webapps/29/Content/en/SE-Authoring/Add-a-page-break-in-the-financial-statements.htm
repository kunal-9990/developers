Add a page break in the financial statements

You can add a page break anywhere in the document to manually mark where a specific page ends and a new one starts. This allows you to control the page layout for printing. You can also use a page break to mark a place where you plan to add content, to create a blank space between two areas or to customize the header or footer for a specific page in the document.

To add a page break:

  1. Locate where you want to add your page break.

  2. Go to the left side and select Add () to add new content.

  3. Choose Page Break from the list of content types.
