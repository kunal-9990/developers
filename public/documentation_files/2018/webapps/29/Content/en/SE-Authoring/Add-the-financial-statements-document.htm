Add the financial statements document

Financial statements are written records that convey the business activities and the financial performance of a company. They include the balance sheet, income statement and the statement of cash flows.

You need to set up the financial statements document to prepare a complete automated solution. Users then find all the statements and notes they need to prepare a compliant set of financial statements document, where only the relevant content for a specific engagement displays.

Once users import the trial balance, the financial statements document is automatically added in the Documents list in the engagement. They can review the document and make some changes to meet specific firm requirements. For information on the changes that users can make in the financial statements document, see What can users modify in the financial statements?

To create the financial statements document, select Add () at the documents page, then choose Financial Statements. A blank financial statements document opens and you can now start authoring.
