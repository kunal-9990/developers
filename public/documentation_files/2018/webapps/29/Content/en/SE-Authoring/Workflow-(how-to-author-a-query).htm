Workflow (how to author a query?)

To start authoring a query or a query template, select Edit ().

The following steps summarize the recommended workflow for authoring queries. You can use this workflow for guidance.

  1. Add a query template

  2. Author the query template

    • Add guidance

    • Add groups

    • Review and customize the query settings

    • Add a response type to a query

    • Add a reference to a document

    • Link a query to a checklist procedure

    • Rearrange queries

    • Add visibility settings

  3. Add a query using an existing template
