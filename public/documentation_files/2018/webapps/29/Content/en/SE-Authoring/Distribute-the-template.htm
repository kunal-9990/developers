Distribute the template

To distribute the template, you need to save the draft you’ve created.

To save your draft:

  1. Navigate to the Cloud menu, then select Settings.

  2. Select your product name, then select Manage Template.

  3. Select Save as to create a template that’s ready for distribution and use.

    4. The template version and creation date display on the page under Draft Version and Publish Date respectively.

Best Practice: We recommend that you test your template before you distribute it.

To test the template, create an engagement file and go through the workflow to make sure everything you’ve configured works as expected. For example,

  • visibility logic works as expected.

  • the trial balance import is successful. Groups and accounts are correctly mapped.

  • the financial statements document populates with data correctly. You can successfully add the Statement of Cash Flows.

  • settings related to fonts, decimals, etc. are correct.

  • placeholders, dynamic text and glossary items are displayed correctly where you defined them.

To distribute the template, you need to export it then contact Third Party Solutions.

To export the template:

  1. Navigate to the Cloud menu, then select Settings.

  2. Select your product name, then select Manage Template.

  3. Select Export to generate the *.se file.

  4. Contact Third Party Solutions.

You can also send the *.se to CaseWare for troubleshooting if needed.
