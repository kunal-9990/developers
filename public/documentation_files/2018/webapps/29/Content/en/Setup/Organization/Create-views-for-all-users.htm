Create views for all users

Using the Views feature, you can set which columns appear in each Cloud page for contacts and staff by default. Cloud comes with predefined views, but you can also create new views and assign them as needed. You can also give users the option to switch between multiple available views.

Create a new view

You can create new views from Settings.

To create a new view:

  1. Ensure that you have the Admin role.

  2. From the Cloud menu, select Settings.

  3. Select Customization | Views Management.

  4. Select New and then choose a Cloud page.


  5. In the editor, enter a Name and select who can access the view (Contacts, Staff or both)

  6. To add new columns, select Add Columns.


    Tip: You can reorder columns using the reorder icon ().

  7. Set the column width from the Width drop-down and choose a sort order.

  8. Modify filter settings in the Filters sidebar. Any filters you add will be automatically applied to the view you create.

  9. Select Save.

The View is added, and can now be assigned to the desired user types.

Assign views

After you have created a view, you will need to configure what types of users can access it or have it set as a default.

To set View access:

  1. Ensure that you have the Admin role.

  2. From the Cloud menu, select Settings.

  3. Select Customization | Views Management.

  4. Select a view from the list.

  5. You can modify the following properties:

    • Default view

    • Overall availability


      The checkmarks on the right indicate which Views are Predefined (these views cannot be edited or deleted), and which have been set as the staff and contact defaults.

You have set the default and available views for staff and contacts.
