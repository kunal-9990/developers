Known issues in CaseWare Cloud

This topic describes known issues when using CaseWare Cloud.

Errors with removed Time users

Currently, users who are logged in and have their Time access revoked by an administrator will get multiple error messages when they attempt to access Time.

Error messages on the Working Papers page

Inaccurate error messages are sometimes displayed when abandoning a sync copy or when clearing sync information on the Working Papers page. To work around this issue, wait a few minutes and then refresh the page to confirm the intended action took place.
