Client requests

  • Creating a client request
  • Sending a client request

This feature is only available with products on the CaseWareCloud SE platform.

Learn more about Smart Engagement products.

When your engagement tasks require information from your clients, you can create client requests from your engagement file. You can use client requests for a number of different things:

  • Request for information - submit questions to the client.

  • Document request - ask the client to upload one or more files.

  • Send deliverables - send files to the client.

Client requests can be accessed by any clients who have a contact profile in Cloud and who have visibility to their engagement.

Creating a client request

Client requests are a type of document for Smart Engagement. When you want to request information from your client, you must first add a new client request to the engagement file.

To add a client request to the engagement file:

  1. From the top menu, select Documents.

  2. Select the Add Document icon (), then select Query.

  3. Select a query type, and enter a name.

    4. Note: Available query types will differ depending on your Smart Engagement product.

  4. Select Create Query.

You have created a new client request document. The client request is displayed, and you can begin reviewing and modifying the client request as necessary. If you created your client request from a template, then you may only need to make minor changes.

Note: If you have created a blank client request, or if you are making major changes to your document, see Modifying a client request for more information.

Sending a client request

Once you have reviewed the client request and made any necessary changes, you can send the document to your client.

To send a client request to a client:

  1. At the top of the document, select the status drop-down menu, and change the status to Open.

  2. In the Open Query dialog, add more details about the client request. You can set a due date, choose assignees from your client contacts, and add a message to the notification email.

  3. Select OK to submit the client request.

Once the query is open, your client contacts can log in to Cloud and see the client request in the My Work section of their overview page.
