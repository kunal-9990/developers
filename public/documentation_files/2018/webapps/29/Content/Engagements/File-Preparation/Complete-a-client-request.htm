This feature is only available with products on the CaseWareCloud SE platform. Available SE products include OnPoint PCR — Preparation, Compilation and Reviews and Fusion RCT — Review, Compilation and Tax.

Responding to a client request

Once you have shared your client request, your contact can see the request on the My Work pane when they log in to Cloud. They can select the client request to display it and enter their responses and comments.

Your contact can mark the client request as Responded by selecting the link at the top of the page.

Note: Contacts can also mark individual procedures as Responded by selecting the Responded icon ().

Once your contact has responded to the request, you will see an icon beside the client request on your list of documents.

Reopening a client request

Select the document to open it and to review your client’s responses. If you need more information or corrections on the document, you can Reopen the client request.

To reopen a client request:

  1. From the top menu, select Documents.

  2. Select the client request from the document list.

  3. In the document, select the status drop-down menu, then choose Reopened.

If there are only problems with some of your client's responses, you can also reopen individual procedures.

To reopen a query procedure:

Select the Prepared icon () beside the procedure, and then choose Decline.

Once the client request has been reopened, the contact for your client should see the new status. They can access the client request from Cloud again to address your comments or to add missing information.

Accepting a client request

If the client has provided all of the requested information, you can change the status to Complete.

To accept a client request:

  1. From the top menu, select Documents.

  2. Select the client request from the document list.

  3. In the document, select the status drop-down menu, then choose Complete.

The client request is now accepted. The information provided in the document is finalized and can no longer be changed. Additionally, completed client requests are no longer visible to contacts in Cloud.
