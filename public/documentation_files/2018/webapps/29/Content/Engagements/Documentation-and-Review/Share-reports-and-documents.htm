You can share files with staff members or contacts in Cloud by assigning roles on the file. For example, assume you want a client contact to be able to view a file in Cloud to verify that its contents are up to date. You would assign them the Contact - Viewer role or higher to ensure that they can access the file and provide feedback on its contents.

If the person you are sharing with doesn't have access to Cloud, you can simply download the files to share in person or via email.

Sharing files with a user or group

You can assign roles on a file or activity from the Files page.

To assign roles on files or activities to a user or group:

  1. Ensure that you have the Owner role or equivalent rights to the file or activity.

  2. Select the files or activities you want to assign roles for, then select Share ().


  3. Select the users or groups you want to have access to the file or activity. Select Assign Roles to (#) and the appropriate roles.

    Any roles that users already have due to previous settings appear under Inherited.

  4. Select Share.

You have shared the file with the selected users.

Downloading a single file from Cloud

If you need to be able to access or share a file outside of Cloud, you have the option to download it to your computer.

To download a single file:

  1. Ensure that you have the Entity Access role or equivalent privileges.

  2. Select the file you want to download.

  3. Select More Actions () | Download from the fileâ€™s menu.

You have downloaded the file to your computer.

Downloading multiple files from Cloud

If you have access to multiple files that you want to download at once, you can select multiple files from the Files page and download them as a .zip package.

To download multiple files as a .zip:

  1. Ensure that you have the Viewer role for the files you want to download or equivalent privileges.

  2. Select the files you want to download and compress into a zip file.

  3. In the Selected File Summary pane, select the More Actions () | Download.

  4. In the Download dialog, enter a name for the zip file and select OK.

     

The selected files are downloaded to your computer in .zip format.
