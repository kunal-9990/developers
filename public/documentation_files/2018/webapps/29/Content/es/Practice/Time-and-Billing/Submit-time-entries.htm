Submit time entries

This feature is only available with CaseWareCloud Time, part of the CaseWare Cloud suite.

Learn more about Time.

To accurately record your own time and expense entries, take advantage of the Time and Billing apps. These apps enable you to submit entries from any browser, regardless of location.

Submitting your own time entries

Submit time entries to record your work using the Time app. For example, you could create a time entry immediately after visiting a client, ensuring that the entry is more accurate than one completed at a later date.

To submit your own time entries:

  1. Entering billable time for an entity requires the Entity Access role for that entity (or the equivalent privileges) and the Process own entries permission.

  2. From the Cloud menu, select Time.

  3. Select New.

  4. Enter the applicable details. A work code and a client name or number are required.

  5. Select Save.

An editable time entry is displayed in the Time app.

Submitting your own bulk time entries

Submit your own bulk time entries using the Time app.

To submit your own bulk time entries:

  1. Entering billable time for an entity in bulk requires the Entity Access role (or the equivalent privileges) and the Process own entries permission.

  2. From the Cloud menu, select Time.

  3. Select New.

  4. Select the Bulk tab.

  5. Choose the applicable Workdays. Total hours will be divided equally between the chosen days.

  6. Enter the applicable details. A work code and a client name or number are required.

  7. Select Save.

Editable time entries are displayed in the Time app.
