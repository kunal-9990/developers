Cloud security architecture

  • Where is my data stored?
  • How is my data protected?
  • Is my data backed up?

CaseWare takes the security of your data very seriously. Below, you can find information about the security structure for Cloud.

Where is my data stored?

Your data is stored in data centers operated by Amazon Web Services. These servers may be located outside of the country you are working from. CaseWare Cloud utilizes Amazon Web Servers in the United States for the Americas, in Australia for Asia-Pacific clients, and in Ireland for other jurisdictions. All of your data, including your backup data, is in the same region.

How is my data protected?

Amazon provides physical security and logical security controls at the infrastructure layer. The platform is covered by a SSAE 16 report (by Ernst and Young) and is PCI Level 1 certified, ISO 27001 certified, and compliant with all major security control frameworks.

CaseWare Cloud uses AES 256-bit encryption keys - the same strength used in online banking. The encryption keys are generated and owned by CaseWare Cloud Ltd., and never shared with other parties, including Amazon, or employees of CaseWare Cloud Ltd. The same level of encryption is applied to:

  • data transferred to and from CaseWare Cloud

  • data at rest

  • backup data

Is my data backed up?

Yes. Your data is backed up daily, and each backup is maintained for 80 days.

CaseWare Cloud utilizes Amazon Glacier for backup storage. Glacier data is specific to a single region and it is not automatically replicated anywhere else. If you have legal requirements that your data must be stored geographically in one region, that data will only be stored there.
