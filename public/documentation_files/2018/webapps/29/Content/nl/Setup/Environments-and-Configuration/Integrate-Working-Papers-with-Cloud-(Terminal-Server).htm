Working Papers integreren met Cloud

Een integratie met CaseWare Cloud biedt extra mogelijkheden in het gebruik van Working Papers. Door de twee producten samen te gebruiken kunt u Cloud als bestandsbeheer en opslagplaats gebruiken en het biedt nieuwe mogelijkheden om uw opdracht online te volgen en af te ronden. Als de twee zijn geïntegreerd in een Terminal Server omgeving, kunnen alle medewerkers die de applicatie vanaf de server starten inloggen met hun individuele Cloud-inloggegevens.

Working Papers integreren met Cloud:

  1. Zorg ervoor dat u Working Papers 2016 of hoger geïnstalleerd heeft.

  2. Start het Working Papers programma.

  3. In het Bestand menu, kies Opties | CaseWare Cloud.

  4. Zorg dat het vakje Integreer met CaseWare Cloud is aangevinkt. Vul bij Server de URL in van de CaseWare Cloud site die uw bedrijf gebruikt. U hoeft het webprotocol niet in te vullen (http:// of https://). Klik op OK.

  5. Er verschijnt een CaseWare Cloud Login venster in beeld. Voer uw Gebruikersnaam en Wachtwoord in en klik dan op OK.

Uw versie van Working Papers is nu geïntegreerd met Cloud.

Een standaard Cloud-Url opzetten voor uw personeel

Iedere medewerker die de applicatie start vanaf de Terminal Server dient zijn of haar unieke inloggegevens en de Cloud-URL in te voeren om verbinding te kunnen maken met Cloud. Door gebruik te maken van een script kunt u de registerinstellingen van de medewerkers aanpassen om automatisch uw Cloud-URL gegevens in te vullen.

De standaard-URL voor uw personeel instellen:

U kunt de registersleutel hier vinden:

  • HKEY_CURRENT_USER\SOFTWARE\CaseWare International\Working Papers\2016.00\Core

  1. Klik met de rechtermuisknop op de Server sleutel en klik dan op Aanpassen....

  2. In het veld voor de Waardegegevens vult u uw Cloud-URL in. Laat het webprotocol (http:// of https://) weg uit de URL.

  3. Klik op OK om uw veranderingen op te slaan.

Medewerkers die het menu Opties | CaseWare Cloud openen zien dat de Cloud-URL ingevuld is. Zij kunnen hun logingegevens invoeren om de integratie te voltooien.
