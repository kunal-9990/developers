Working Papers koppelen met Cloud:

Deze functie is alleen beschikbaar bij gebruik van CaseWare Working Papers.

Meer informatie over CaseWare Working Papers.

Working Papers koppelen aan CaseWare Cloud biedt extra mogelijkheden. Door de twee producten samen te gebruiken kunt u Cloud als bestandsbeheer en opslagplaats gebruiken en het biedt nieuwe mogelijkheden om uw opdracht online te volgen en af te ronden.

Als de twee zijn gekoppeld in een Terminal Server omgeving, kunnen alle medewerkers die de applicatie vanaf de server starten inloggen met hun individuele Cloud-inloggegevens.

Working Papers koppelen met Cloud:

  1. Open Working Papers.

  2. Onder de tab Bestand, kies Opties | CaseWare Cloud.

  3. Zorg dat het vakje Integreer met CaseWare Cloud is aangevinkt. Vul bij Server de URL in van de CaseWare Cloud site die uw bedrijf gebruikt. U hoeft het webprotocol niet in te vullen (http:// of https://). Klik op OK.

  4. Er verschijnt een CaseWare Cloud Login venster in beeld. Voer uw Gebruikersnaam en Wachtwoord in en klik dan op OK.

Uw versie van Working Papers is nu gekoppeld met Cloud. Als u Working Papers gebruikt in een omgeving met een terminal server dan kunt u een standaard Cloud-URL opgeven voor uw organisatie. Dit bespaart tijd bij het inloggen.

Een standaard Cloud-Url opzetten voor uw personeel

Als u werkt in een omgeving met een een terminal server moeten uw medewerkers iedere keer als ze Working Papers openen hun gegevens invoeren, inclusief een Cloud-URL. Om hier tijd op te besparen kunt u een standaard Cloud-URL instellen die automatisch wordt ingevoerd als Working Papers wordt geopend.

De standaard URL voor Cloud instellen in Working Papers:

U kunt de registersleutel hier vinden:

  • HKEY_CURRENT_USER\SOFTWARE\CaseWare International\Working Papers\2016.00\Core

  1. Klik met de rechtermuisknop op de Server sleutel en klik dan op Aanpassen....

  2. In het veld voor de Waardegegevens vult u uw Cloud-URL in. Laat het webprotocol (http:// of https://) weg uit de URL.

  3. Klik op OK om uw veranderingen op te slaan.

De standaard Cloud-URL is nu ingesteld.
