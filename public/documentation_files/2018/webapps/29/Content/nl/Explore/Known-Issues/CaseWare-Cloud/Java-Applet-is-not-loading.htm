De Java-app laadt niet

Als het laadscherm lang blijft hangen kan het zijn dat de plug-in van Java uitgeschakeld is.

Volg de instructies die op de Java website staan voor oplossingen voor uw browser.

Het kan ook gebeuren dat de Java-app niet werkt. Als u hulp nodig heeft neem dan contact op met CaseWare support of uw distributeur. Deze issue wordt verholpen in een toekomstige release.
