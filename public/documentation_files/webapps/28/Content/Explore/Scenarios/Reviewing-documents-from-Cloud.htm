Review on the Go is a feature that enables you - as a reviewer - to conveniently verify the accuracy and quality of documents from any browser on your favorite mobile or tablet device. Anyone with access to the Working Papers engagement file can share individual documents to Cloud, making it convenient for partners or senior staff to review documents and provide instant feedback, without having to install Working Papers, pass documents back and forth via email, or attend numerous meetings.

Reviewers can make changes to any document and create review notes for the engagement team to resolve. The feedback is instantaneous and merged directly into the Working Papers engagement file. Team members that have the Working Papers file open and in an "online" state, automatically receive the changes from the reviewer.

The reverse situation is not true, however. If an engagement team member makes a change to a document that was already sent for review, then the reviewer will not be able to see the specific update; only an indication that the document has been modified. Only by re-sending the document for review, can the reviewer see the specific updates to the file.

Set up Working Papers on Cloud

In order to start using Review on the Go, you need to enable Working Papers integration with CaseWare Cloud. For more information about integration, see Integrate your Working Papers with Cloud.

Once the connection is established, ensure to create a primary (parent) copy on Cloud so that your engagement team can all work in the file at the same time. This process is known as publishing the file. After the file has been published, your team can create sync copies to begin working in the file together.

Next, ensure role sets are defined so that you can track which roles have signed off on the document.

Create and modify role sets

Review on the Go uses the role sets defined in the Working Papers file. You can add, modify or delete a role set to be used in the Review on the Go session.

To create and modify roles sets:

  1. In Working Papers, open the Tools | Options dialog.

  2. Select Roles.

  3. Click Add ( ) to add a new role set or Edit () to edit an existing role set. To delete an existing role set, click Delete ( ).

  4. Specify the Number of Roles for this set.

  5. For each role, specify its:

    • Label - Select the appropriate label (Prepared by or Reviewed by) or enter a custom label in the text box.

    • Requires - Use the drop-down to specify if other role sign-offs are required prior to signing off this role. For more information, see Role Dependencies.

    • Color - Select the color of the checkmark used on the Document Manager when this role is signed off.

  6. Click OK.

You have created, modified, or deleted the role set for your client file. Your engagement team can now use these roles when reviewing files in Review on the Go. Ensure that all document collaborators are synced and have received these changes.

Send documents for review

When you are ready to submit your assigned documents for review, you can use Review on the Go to help manage the process.

Creating a review session from select documents

You can set up a review session using CaseWare® Working Papers. Each review session using Review on the Go creates an active synchronization between the client file and CaseWare Cloud.

To create a review session from select documents:

  1. Open the Working Papers application on your desktop.

  2. Open your sync copy of the file to review.

  3. On the Cloud tab, in the Review on the Go section, click Setup to access the setup wizard.

  4. Click Next to map Working Papers User Names to the corresponding Cloud User Names. Use the drop-down menu to select a Cloud user from the list.

  5. Click Next to select the documents for this review session.

  6. Select the documents or folders that are ready for review and click Next.

  7. Click Next to complete the setup wizard.

  8. Click Finish.

  9. After your upload is complete, a link to the review session is provided. Select Create an Activity for this action to automatically post an Activity to Cloud. This will notify other staff of this review session.

  10. Click OK to complete the setup.

 

You have created a new review session in Review on the Go. Other members of your engagement team can now review your selected documents.

Signing-off on the preparer Role

If you forgot to sign off on your role after creating a review session, you can do so by entering the review session and selecting the applicable role.

To sign off on a document:

  1. Select the applicable role.

  2. Select Save.

  3. Select the appropriate documents.

  4. Select the checkmark to sign off on the documents in bulk and select Sign-off in the Signing off Selected Documents dialog.

  5. Select the Gear icon( ) | Sign Out to exit the review session.

You have signed off on a document in the Review on the Go session.

Notifying your reviewers

Once your documents are ready for review, you can send a notification email to the reviewer and limit the activity to the reviewer. You can do this in Cloud by modifying the activity you created when you created the Review on the Go session.

To modify an existing activity:

  1. Open the Activities app.

  2. Select the activity that needs to be modified.

  3. In the pane to the right, select Edit ().

  1. Edit the description and mention your reviewer by using the @ symbol. Type the name of the person or entity to initiate the look-up list. Press Esc to stop searching.

  2. Select the Limited slider to limit access to this activity to only the reviewer and select Save and Share to open the Share dialog.

  3. In the Share Dialog, locate your reviewer and grant access to the reviewer.

  4. Select Share.

You have mentioned your reviewer on the Review on the Go activity. The reviewer will receive a notification email from Cloud, and they will be able to start reviewing your work.

Review documents

Once the preparer has created a review session, you can view documents from a client file using Review on the Go. Reviewers can access these documents by clicking on the attachment shared by the preparer.

These documents can be updated by the engagement team during a review session. Reviewers and engagement teams should ensure the document submitted for review are always up-to-date.

If the preparer did not create an activity, you can access the latest submitted documents from the file from the Working Papers page and adding the Review on the Go filter.

Select the reviewer sign-off role

Reviewers select the role they are signing off as soon as they open Review on the Go. You can change the role sign off at any time.

To select your sign-off role:

  1. Open the review session.

  2. Select the Settings icon () at the top of the page.

  3. Select the applicable reviewer role for sign off.

  4. Select Save.

Create and assign a review note

You can create a review note directly in Review on the Go. These review notes are automatically synchronized to the client file and visible to all engagement team members when they open the file.

To create and assign a review note:

  1. In the document being reviewed, select the Create an Issue button ()

  2. Complete the review note details.

  3. Select Save.

You have created a new issue. This new issue will be added to the client file and accessible from both Review on the Go and Working Papers.

Modifying or re-assigning review notes

If you made a mistake, you can always go back to modify the review note. In the case above, our reviewer forgot to assign the review note to the preparer.

To modify or re-assign a review note:

  1. In Review on the Go, select the Issues tab.

  2. Select the issue name. The issue details will appear in a new pane.

  3. Make your modifications and select Save.

The modifications are saved and the review note is now assigned to the preparer.

Sign off on documents

Once you have completed your review, you can sign-off on the document to notify the team that the documents have been reviewed.

To sign off on a document:

  1. Select the applicable role.

  2. Select Save.

  3. Select the appropriate documents.

  4. Select the checkmark to sign off on the documents in bulk and select Sign-off in the Signing off Selected Documents dialog.

  5. Select the Gear icon( ) | Sign Out to exit the review session.

You have signed off on the selected documents. These documents are updated in Review on the Go and Working Papers.

Re-submit documents for review

Engagement team members can add new snapshots or update existing snapshots during a review session. Reviewers can then receive the latest state of a document or receive new documents that are added to the review.

To re-submit documents for review:

  1. Open a synchronized copy of the file in Working Papers.

  2. On the Cloud tab, in the Review on the Go section, click Snapshot to access the setup wizard.

  3. Select the documents you want to update. Documents with a snapshot have a date stamp that indicates the last time a snapshot of those documents was taken

  4. Click OK.

A new snapshot for each of the selected documents is taken and uploaded by Review on the Go. Reviewers can refresh their browser page to receive the latest snapshots.

End a review session

Once all documents in the session have been reviewed and signed off, you can end the review session.

To end a review session:

  1. Open a synchronized copy of the file in Working Papers.

  2. On the Cloud tab, in the Review on the Go section, click End.

  3. In the notification dialog, click Yes to confirm.
